# 河海大学未来技术学院诚聘英才

## 

发布时间：2024-04-25文章来源：河海大学未来技术学院

**学校简介**

河海大学是一所拥有百余年办学历史，以水利为特色、工科为主、多学科协调发展的教育部直属全国重点大学，是实施国家“211工程”重点建设、国家优势学科创新平台建设、“双一流”建设以及教育部批准设立研究生院的高校。水利工程、环境科学与工程2个学科入选国家一流学科；在学科评估中，水利工程、环境科学与工程、土木工程保持领先，工程学、环境/生态学等9个学科进入ESI排名全球前百分之一，其中工程学进入ESI全球前千分之一。

**学院概况**

未来技术学院于2023年批准设立，聘请澳大利亚工程院院士谢亿民教授担任名誉院长。学院聚焦国家重大战略和世界科技前沿，瞄准土木水利行业未来10-15年的前沿性、革命性、颠覆性技术，凝练交叉学科方向，探索未来技术领军人才培养模式，建设服务国家战略和地方经济发展的科技资源共享服务平台。学院拥有“十三五”江苏省“双创计划”人才团队和“十四五”江苏省千万级高层次人才团队，设有“江苏省智能安全与结构优化”院士协同创新中心，支撑学校多个重大平台建设。重点开展：

（1）聚焦数字水利、未来结构设计、机器人建造三个方向，着力解决“智能制造”和“智能建造”相关的参数化建模、数值仿真、检测监测、结构优化以及碳纤维等新材料、增材制造、人机交互、机器人建造等关键技术问题。 

（2）依托国家“新工科”战略和常州市“龙城科创学院”，选择智能制造工程（未来技术方向）本科专业开展项目制人才培养，探索形成满足未来技术发展需要的创新创业人才培养体系。

（3）瞄准国家重大需求、结合地方产业特色，打造机器人建造方向前沿技术科技资源平台，并产生标志性成果。

2023年，学院共获得江苏省及常州市各类专项经费支持5000万元，科研成果在国家下一代舰艇设计平台、长江奉节二桥等重大工程中获得运用。

根据学院发展需要，现面向全球诚聘英才：

**引聘对象**

1.专业方向：土木工程、水利工程、机械工程等专业的结构工程方向、增材制造方向、数字孪生方向，智能制造和机器人方向等；

2.国内外高水平大学或研究机构的教授、副教授（或相当职称）或研究团队；

3.海内外知名高校博士毕业生、博士后出站人员。

**相关待遇**

**高额薪酬待遇：**提供有竞争力的薪酬福利待遇，可申请学校“大禹学者”人才培育支持计划，入选者在支持期内可额外享受专项津贴。

**解决事业编制：**教授、副教授，结合年龄等实际情况可按事业编制用人方式聘用。 

**提供科研支持：**提供有竞争力的平台建设经费、科研启动经费、良好的实验和办公环境。

**保障团队建设：**根据团队建设需求可给予一定的进人指标和研究生招生指标。

**助力人才安居：**提供住房或住房补贴、安家费，过渡期内可提供教师周转公寓等。协助办理落户、外国人永久居留证等。

**畅通发展通道：**大力支持符合条件的人才依托我校申报国家级人才计划，以及江苏省“333工程”、江苏省“双创计划”等省级人才工程/计划。

**家属待遇优厚：**保证子女进入河海大学幼儿园（江苏省省级示范园），协助解决子女中小学优质教育资源和配偶工作。

**便捷生活服务：**可享受南京市“紫金山英才计划”、常州“龙城英才计划”等地方政府政策，享受医疗、购房、家政、出行等方面的优惠待遇。

**应聘材料**

请有意向应聘者提供个人简历（含个人基本信息、受教育情况、主要论文论著、科研和获奖情况等）。
