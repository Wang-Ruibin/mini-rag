# 河海大学合作处、校友办、基金办赴华东理工大学、东华大学开展专题调研

> 来源: https://hhuedf.hhu.edu.cn/2026/0708/c26739a331204/page.htm

7月6日—7日，河海大学国内合作发展处处长、校友会办公室主任、教育基金会办公室主任朱乔带队赴上海，先后走访华东理工大学校友与发展联络处、东华大学发展联络处，围绕基金、校友及合作工作开展专题调研。

在华东理工大学调研期间，该校校友与发展联络处处长、校友总会秘书长、校基金会秘书长王芝眉介绍了校董会、教育发展基金会、校友会的发展概况。双方聚焦校院两级筹资体系搭建，筹资工作推进过程中的重点难点问题、创新破解路径进行深入交流，并就校友资源挖掘整合，校友及合作赋能筹资工作等方面交流思路方法。

在东华大学调研期间，该校发展联络处处长、基金会秘书长、校友会秘书长沈洁介绍了校友会、基金会的发展现状，重点展示基金会特色工作、公益品牌活动开展、社会资源联动等方面工作成果与创新实践。双方立足新时代高校教育基金会发展新形势、新要求，就校院两级筹资体系建设、筹资机制创新、校友工作与基金工作协同提质等内容进行了深入交流。

朱乔对两所高校的经验分享表示衷心感谢。他表示，两所高校在校院两级筹资体系构建、校友工作与基金工作深度融合方面的理念和创新做法，都为河海大学提供了宝贵的经验；将充分吸收本次调研学习成果，结合学校办学实际和发展需求，进一步优化筹资工作体制机制，深化校友联络与对外合作内涵，持续提升资源整合能力和社会服务水平，依托高校间常态化联动交流，主动融入和服务长三角教育一体化高质量发展格局。
