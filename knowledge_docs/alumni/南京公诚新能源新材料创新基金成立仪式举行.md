# 南京公诚新能源新材料创新基金成立仪式举行

> 来源: https://hhuedf.hhu.edu.cn/2026/0320/c26740a323774/page.htm

3月18日，南京公诚新能源新材料创新基金成立仪式在河海大学常州校区举行。南京公诚节能新材料研究院、金坛区科技局相关负责人，河海大学国内合作发展处、常州校区管委会负责人，新能源学院、材料科学与工程学院负责人和师生代表参加。活动由新能源学院党委书记杨春伟主持。常州校区管委会主任吴永强致欢迎词。金坛区科技局副局长刘娅敏介绍校地合作情况。

南京公诚节能新材料研究院董事长孙俊杰介绍了企业发展历程和发展成就，表示将持续深耕新能源新材料领域，全力支持河海大学科技创新与人才培育工作，推动更多科研成果实现产业化落地。

河海大学国内合作发展处处长、教育基金会办公室主任朱乔对南京公诚节能新材料研究院的捐赠善举致以诚挚谢意，期待校企双方发挥各自优势、实现资源互补，全力推动科研成果转化，助力创新价值与产业价值双向赋能。朱乔与孙俊杰共同签署捐赠协议，并代表学校向孙俊杰颁发了捐赠证书、捐赠鸣谢牌。
