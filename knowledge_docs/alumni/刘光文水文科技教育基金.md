# 刘光文水文科技教育基金

> 来源: https://hhuedf.hhu.edu.cn/2026/0205/c26741a322953/page.htm

刘光文水文科技教育基金于2010年设立，从2011年开始在全国范围评审，在国内水文乃至水利行业产生了较大社会影响。该基金分设“刘光文科技奖”“刘光文奖学金”，2025年起，“刘光文科技奖”包括“刘光文科技成就奖”“刘光文国际合作奖”“刘光文工程技术奖”“刘光文青年科技奖”，用于奖励在我国水文行业作出杰出贡献的科研人员、工程技术人员、管理人员、高校教师和优秀青年科技工作者，每两年评选一次；“刘光文奖学金”用于奖励我国高等院校水文专业优秀学生，每年评选一次。
