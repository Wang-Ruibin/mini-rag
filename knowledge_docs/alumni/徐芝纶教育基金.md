# 徐芝纶教育基金

> 来源: https://hhuedf.hhu.edu.cn/2026/0205/c26741a322952/page.htm

徐芝纶教育基金于1996年设立，由徐芝纶院士及其夫人伍玉贤、家属、河海教师共同出资设立了“徐芝纶教育基金”，用于奖励优秀教师、优秀学生、力学工作者和优秀教材。2005年在江苏省力学学会的发起下，面向全国增设了“徐芝纶力学奖”。2011年为纪念徐芝纶院士诞辰100周年，特设一等奖2名，二等奖5名。自2015年第六届开始调整为2名，不分等级。多名获奖者在获奖后当选院士，受到了全国力学界的广泛关注和认同。目前设有“徐芝纶教学奖”“徐芝纶力学奖”“徐芝纶奖学金”三个奖项。
