# 信息科学与工程学院举行南京戎光软件科技有限公司国创赛孵化金捐赠仪式

> 来源: https://hhuedf.hhu.edu.cn/2026/0611/c26739a327814/page.htm

6月5日，南京戎光软件科技有限公司国创赛孵化金捐赠仪式在河海大学常州校区21号楼101报告厅隆重举行。河海大学副校长邓成，南京戎光软件科技有限公司董事长孙晓波、副总经理张万君、总经理助理蔡琳宇，江苏河澥海浔物联科技有限公司总经理胡文育，信息科学与工程学院党委书记张伟建、副院长霍冠英、党委副书记胡井军和百余名师生代表参加了此次仪式。

邓成对南京戎光软件科技有限公司热心教育事业、助力青年科创的善举表示谢意。他指出，南京戎光软件深耕国防信息化、数字孪生研发领域，科研实力突出、富有社会责任感，此次在信息院设立国创赛孵化金，既是企业创新精神与科创情怀的生动彰显，更是校企同向同行、协同育人、服务国家创新驱动发展战略的生动实践。学校将以此次捐赠为契机，进一步完善科创人才培育体系，常态化开展项目指导、团队孵化、成果打磨、赛事培优，最大化发挥资金育人效能，让更多青年学子的创新想法落地为优质项目，让更多校园科创成果走出校园、走向赛场、对接产业。同时，他勉励青年学子心系家国、苦练专业本领、常怀感恩之心，借力校企优质资源锤炼创新能力。

孙晓波对河海大学的信任与邀约表达感谢。他表示，公司将以赛事为纽带发掘优质科创项目，从技术、产业等层面为参赛学生提供全方位支持。他期待校企双方能够围绕课程共建、科学研究、人才培养等深化产学研合作，取得更多务实成果，助力双方在服务国防信息化建设与国家科技创新发展的进程中实现共赢。

邓成为孙晓波颁发了捐赠证书与鸣谢牌。

仪式结束后，孙晓波董事长作题为《十二载军工科创创业路：从 BIM到AI数字孪生的产业深耕之旅》的创业故事分享。
