# 河海大学教育发展基金会召开第四届理事会第七次会议

> 来源: https://hhuedf.hhu.edu.cn/2026/0424/c26740a325686/page.htm

4月21日，河海大学教育发展基金会召开第四届理事会第七次会议。会议由基金会理事长郭继超主持，基金会理事、监事出席会议。

会议听取并审议了基金会2025年度财务工作报告、2026年工作计划、财务预决算、捐赠资金使用方案及开展陕西省石泉县乡村振兴帮扶项目等议题。与会理事围绕资源拓展、公益项目策划及实施、内部规范治理、提升服务质效等方面进行深入交流，为基金会助力学校高质量发展建言献策。郭继超理事长在讲话中强调，基金会要坚守公益初心，紧扣学校高质量发展与“十五五”规划开局部署，扎实推进各项工作：一是坚持党的全面领导，完善内部治理体系，严守制度规范，推进等级评估，不断提升规范化水平；二是聚焦资源拓展，梳理重点合作资源，优化捐赠项目体系，推动社会资源与学校发展需求精准对接；三是强化精细管理与专业服务，加快信息化建设，构建捐赠全生命周期服务机制，持续提升服务质效；四是积极打造公益品牌，汇聚社会力量赋能学校高质量发展，充分发挥社会组织公益作用，为学校建设世界一流大学、助力教育强国建设贡献更大力量。
