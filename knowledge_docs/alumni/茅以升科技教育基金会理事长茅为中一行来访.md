# 茅以升科技教育基金会理事长茅为中一行来访

> 来源: https://hhuedf.hhu.edu.cn/2026/0320/c26740a323775/page.htm

3月13日，茅以升科技教育基金会理事长茅为中一行赴河海大学开展座谈。河海大学副校长陈达会见茅为中一行。河海大学国内合作发展处、教育基金会办公室、科技处、苏州高等研究院、土木与交通学院等相关单位负责人及河海大学茅以升班学生代表参加座谈。

国内合作发展处处长、教育基金会办公室主任朱乔对茅为中理事长一行表示热烈欢迎，衷心感谢茅以升科技教育基金会长期以来对学校发展的关心支持。他介绍了学校的发展近况，并指出茅以升先生曾担任河海工科大学校长，其“爱国、科学、奋斗、奉献”的精神超越时空、历久弥新，“茅以升科学技术奖”系列奖项影响力广泛，为高校人才培养与学科发展搭建了优质平台。今年恰逢茅以升先生诞辰130周年，学校愿与茅以升科技教育基金会进一步深化人才培养、科技创新、社会服务等领域合作，传承弘扬茅以升精神，助力科技自立自强与教育强国建设。茅以升科技教育基金会理事长茅为中对学校在传承与弘扬茅以升精神方面所做的工作表示感谢，并表明茅以升科技教育基金会将继续助力河海大学的发展，重点支持“河海大学茅以升班”建设，在拔尖创新人才培养、工程教育实践、科技公益服务等方面拓宽合作空间，携手培育心怀家国、深耕工程科技的高素质人才，服务国家发展战略。会上，茅以升科技教育基金会副理事长刘文杰介绍了茅以升诞辰130周年系列活动安排，副秘书长杨茜雯介绍了茅以升科学技术颁奖大会等相关筹备工作。河海大学苏州高等研究院院长陈永辉、土木与交通学院党委书记汪建新、科技处副处长张晓祥、土木与交通学院党委副书记安杰晶、河海大学茅以升班班长王自强分别围绕科学研究、成果转化、奖项申报以及茅以升班建设等与茅以升科技教育基金会进行了交流。
