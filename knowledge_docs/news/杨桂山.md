# 杨桂山

## 

发布时间：2024-10-31文章来源：河海大学

杨桂山，男，中共党员，1965年8月出生，江苏兴化人，博士，研究员、博士生导师。现任河海大学党委书记。

1987年7月南京大学地理系自然资源专业本科毕业，随后进入中国科学院南京地理与湖泊研究所工作，1998年7月晋升研究员，2004年12月任所长。2014年7月进入中国科学院南京分院工作，历任副院长（正局级），院长、分党组成员，分党组书记兼院长。2023年4月任河海大学校长、党委副书记。

主要研究方向为流域地理学和资源利用与生态保护，长期致力于流域水土资源利用与生态环境影响及调控研究，多次主持国家973计划和中国科学院战略科技先导专项等重大项目研究。
