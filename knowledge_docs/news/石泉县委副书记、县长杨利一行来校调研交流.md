# 石泉县委副书记、县长杨利一行来校调研交流

## 

发布时间：2026-05-14文章来源：国内合作发展处

5月13日，石泉县委副书记、县长杨利率石泉县代表团来校调研交流。副校长邓成出席座谈会。

邓成表示，定点帮扶石泉是学校的重要政治责任，自开展帮扶工作以来，学校连续选派多名挂职干部，全方位推进产业发展、教育提升、人才培养、水利支持等帮扶工作，为石泉县乡村振兴注入坚实动力。学校将深入落实党中央、国务院及教育部关于乡村振兴工作的有关部署，立足常态化帮扶新阶段，持续发挥教育、科技、人才优势，不断提升帮扶质效，以更高水平助力石泉县全面推进乡村振兴，推动校地帮扶合作向更深层次迈进。

杨利介绍了石泉县以乡村振兴为统领，在县域经济、特色产业、民生改善和生态保护等方面取得的发展成效。他表示，河海大学定点帮扶成果丰硕，在产业振兴、人才振兴、文化振兴、生态振兴、组织振兴各方面‌为石泉县乡村振兴提供了强大助力，希望未来双方持续深化定点帮扶合作，聚焦产业升级、水利赋能、生态保护、智力帮扶等重点方向，携手推动石泉县乡村振兴取得新突破。

调研期间，杨利一行参观了牛首山科技园、校史馆，走访了学校重点合作单位中国电建华东区域总部。

石泉县委常委、政法委书记蔡英宏，副县长黄冬梅以及相关部门负责人，学校相关部门负责人以及在石泉县挂职干部参加了调研活动。
