# 陆国宾

## 

发布时间：2021-09-15文章来源：河海大学

陆国宾，男，中共党员，1966年11月出生，江苏南通人，副研究员。现任河海大学党委副书记、副校长。

1988年7月河海大学陆地水文专业本科毕业。1998年9月到1999年10月受国家留学基金委委派在英国Sheffield大学作访问学者。2017年8月任河海大学党委常委、副校长，2018年4月到2020年4月任江西省上饶市委常委、副市长（挂职），2024年2月任河海大学党委副书记、副校长。兼任江苏省气象学会副理事长。

先后获“江苏省高等学校优秀共产党员”“江苏省高校实验室管理工作突出贡献奖”等荣誉称号。发表各类论文20余篇，编写专著2部。
