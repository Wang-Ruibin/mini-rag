# “绿茵手拉手 辽苏共成长”校园足球交流互访活动座谈会在我校举行

## 

发布时间：2026-06-30文章来源：体育系

6月29日，“绿茵手拉手 辽苏共成长”校园足球交流互访活动座谈会在我校举行。辽宁省委教育工委专职委员许会峰，江苏省教育厅二级巡视员刘广鸣，校党委副书记、副校长陆国宾出席座谈。

许会峰表示，本次活动不仅搭建起辽苏校园足球常态化交流平台，更为两省拓宽教育合作维度、深化优质生源联合培养等领域深度协作创造了有利契机。希望以此次互访交流为新起点，持续在校园足球共建、优质生源输送、教育资源共享等多领域携手深耕，合力推动辽苏两地教育事业协同高质量发展。

刘广鸣对辽宁代表团来访表示欢迎，并介绍了江苏校园足球在普及推广、人才培养、师资建设、赛事体系及文化培育等方面的实践成效，高度评价了河海大学在足球学科建设、高水平运动队竞赛和校企协同育人中发挥的示范引领作用。

陆国宾表示，学校自建校之初便确立“德立、体健，可以进言学术”的育人理念，目前已形成覆盖公共体育教学、足球本硕学科体系、专业师资队伍、校企协同育人及毕业生高质量就业的一体化足球育人模式，为高校体教融合提供了有益范本。

交流研讨环节，双方围绕校园足球招生、特长生贯通升学、赛事体系等议题展开深入交流，互学互鉴、取长补短。

座谈会前，还参观了校史馆和足球相关史料展陈。
