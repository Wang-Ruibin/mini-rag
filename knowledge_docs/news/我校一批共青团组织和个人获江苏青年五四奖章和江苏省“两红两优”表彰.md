# 我校一批共青团组织和个人获江苏青年五四奖章和江苏省“两红两优”表彰

## 

发布时间：2026-04-30文章来源：团委

在五四青年节到来之际，共青团江苏省委、江苏省人力资源和社会保障厅表彰了第20届江苏青年五四奖章和江苏省五四红旗团委（团支部）、江苏省优秀共青团员、江苏省优秀共青团干部。我校多个团组织和个人获得表彰，再次实现省青年五四奖章和省“两红两优”获奖全覆盖。

水利水电学院教授蒋晟被授予江苏青年五四奖章。蒋晟主要从事水工岩土工程研究，为工程岩土体动力响应智能预测与灾害风险评估提供重要支撑。他坚持立德树人，带领学生奔赴祖国大江大河和重大工程现场，在祖国最需要的地方扎根实践，以山河为课堂、以工程为教材，在知行合一中培根铸魂、启智润心，展现了新时代青年科技工作者服务国家战略、勇担使命的责任担当。

港口海岸与近海工程学院团委被授予江苏省五四红旗团委称号。学院团委认真落实立德树人根本任务，深耕思想引领、夯实组织根基、服务青年成长、主动担当作为；规范开展“三会两制一课”，精心打造“科学家精神”信仰公开课特色品牌；深耕创新创业育人沃土，引领青年学子在“挑战杯”“国创赛”等顶级赛事中勇攀高峰，斩获全国特等奖、全国金奖等多项荣誉。

商学院2023级会计学专业3班团支部被授予江苏省五四红旗团支部称号。支部聚焦思想引领、学业科创、实践志愿、素养提升，系统构建“理论浸润+实景研学+骨干引领”三维学习模式，创新实施《团员成长积分办法》，支部成员累计志愿时长超2000小时，获“挑战杯”全国一等奖、江苏省特等奖等国省级奖项多项，支部战斗堡垒作用充分彰显。

水文水资源学院2023级本科生秦朗、人工智能与自动化学院2023级本科生吴钦涵、马克思主义学院2023级本科生臧雪含等3名同学被授予江苏省优秀共青团员称号。他们在专业学习、社会工作、志愿服务、社会实践、创新创业竞赛等方面表现突出，成果显著，集中展现了新时代河海团员青年有信仰、讲政治、重品行、争先锋、守纪律的良好形象。

水利水电学院团委书记笪步源老师被授予江苏省优秀共青团干部称号。笪步源聚焦思想引领，主讲信仰公开课省级精品示范课，所带学生获中国大学生自强之星；聚力组织建设，所带支部获评全国活力团支部、省五四红旗团支部；聚能科研创新，助力学生科创团队获“挑战杯”全国擂主、国创赛全国银奖，以青春担当书写育人华章。

近年来，河海大学团委始终坚持党建带团建，全面贯彻落实学校党委和上级团组织有关要求，坚持全面从严治团，着力推进基层团组织标准化、规范化建设，促进基层团组织切实履行引领凝聚青年、组织动员青年、联系服务青年的基本职责，不断建设更加充满活力、更加坚强有力的共青团组织。
