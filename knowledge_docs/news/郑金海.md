# 郑金海

## 

发布时间：2021-09-15文章来源：河海大学

郑金海，男，中共党员，1972年2月生，福建莆田人，博士、教授。国家杰出青年科学基金获得者、国家级人才计划入选者、英国皇家工程院杰出访问学者。现任河海大学校长、党委副书记。

1993年7月河海大学港口及航道工程专业本科毕业，1998年9月河海大学海岸工程博士研究生毕业后留校任教，2006年12月受聘教授。2017年8月任河海大学党委常委、副校长。曾在日本京都大学和美国麻省理工学院等高校访问研究。

一直从事港口、航道、海岸及近海工程的教学和科研工作，致力于河口海岸动力学、港口航道开发、海岸带保护修复与防灾减灾的创新研究和工程实践。主持国家重点研发计划、国家自然科学基金重点国际合作等科研项目，获国家级教学成果二等奖和国家科技进步二等奖各2项。
