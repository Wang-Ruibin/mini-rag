# 河海大学代表团出访挪威

## 

发布时间：2026-04-23文章来源：国际合作处

4月17日至19日，副校长宋翔洲率团赴挪威开展访问交流，先后访问了奥斯陆大学、卑尔根大学，进一步拓展与挪威高校及科研机构的合作领域。

访问期间，宋翔洲受邀参加中国-挪威高层海洋科技对话活动，并担任活动科技讨论环节主持人，与中挪双方专家学者围绕气候变化、极地科考、联合国海洋十年等全球海洋治理核心议题展开深入研讨，分享河海大学在相关领域的研究成果与实践经验，积极推动双方在全球海洋治理领域的理念对接与技术协作，为构建中挪海洋科技合作新格局凝聚共识。

在奥斯陆大学，代表团与地球科学系主任Joseph Henry Lacasce教授围绕师生互访、科研合作、学术交流等关键领域展开深入磋商，就搭建常态化交流平台、联合开展学术研讨、推进研究生联合培养等事宜达成初步共识，助力双方优势学科互补发展。

在卑尔根大学，代表团与地球物理研究所所长Tor Eldevik教授、贺圣平高级研究员重点探讨了科研项目合作、青年人才互访等合作机制，并就聚焦气候变化、海洋生态保护与可持续发展等前沿方向开展合作研究达成重要共识，为深化双方实质性合作、拓展合作维度奠定坚实基础。

代表团参观了挪威南森环境与遥感中心，就海洋观测技术发展、数据融合与同化、气候变化背景下海洋关键过程研究以及未来重点合作方向进行了交流，在深化合作意向方面取得积极进展。

访问期间，还开展河海大学海外人才全球招聘宣讲活动，宋翔洲与参会学者、学生亲切交流，详细介绍学校人才引育、科研支持、生活保障等方面情况，积极吸引海外优秀人才加盟河海。

下一步，学校将持续跟进此次出访达成的各项合作共识，推动合作落地见效，助力学校海洋学科建设与国际化发展再上新台阶。
