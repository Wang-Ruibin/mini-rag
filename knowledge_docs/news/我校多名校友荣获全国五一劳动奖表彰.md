# 我校多名校友荣获全国五一劳动奖表彰

## 

发布时间：2026-04-29文章来源：校友会办公室

4月28日，中华全国总工会召开庆祝“五一”国际劳动节暨全国五一劳动奖表彰大会，会上表彰了全国五一劳动奖3024个。其中，全国五一劳动奖状379个、全国五一劳动奖章1462名、全国工人先锋号1183个。据初步统计，**我校校友刘晓波、崔冬、周文理、侯轶群、赵梦杰等荣获全国五一劳动奖章，校友衷平领衔的中交公路规划设计院有限公司绿色低碳交通技术攻关团队荣获全国工人先锋号。**

刘晓波，2001届环境工程专业本科毕业、2004届环境工程专业硕士毕业，现为中国水利水电科学研究院水生态环境研究所所长、党总支书记，兼任水利部水环境监测评价研究中心主任、水利部京津冀水安全保障重点实验室副主任。刘晓波长期深耕河湖水生态环境保护与水利工程优化调控领域，紧扣国家提升大江大河治理能力的重大需求，致力于挖掘水工程的生态环境效益。研发了水源地水质安全保障、水工程水质—水量优化调度、河湖生态流量/水位调控等核心技术，在河湖水文—水质—水生态互馈的理论、协同调控等方面取得系统性创新成果。相关成果应用于三峡、南水北调、引江济淮等30余项重大工程。

崔冬，2004届港口及航道工程专业本科毕业、2007届港口、海岸及近海工程专业硕士毕业，现为上海市水利工程设计研究院有限公司总经理助理、水利二分院院长、副总工程师。崔冬长期从事河口海岸工程的设计咨询与技术研发，立足河口海岸工程领域，承担上海南汇、横沙等多项重大促淤圈围工程，支撑国土空间优化与城市发展安全，服务长三角一体化布局，攻克海塘安全、河口治理等关键技术，形成专利、软件及论文群，为沿海防灾与生态韧性提供核心支撑。

周文理，2006届给排水工程本科毕业，2009届市政工程硕士毕业，现为住房城乡建设部村镇建设司一级调研员。周文理长期从事传统村落、农村人居环境整治等村镇建设工作，推动实施传统村落特色保护区建设，保护中华民族宝贵文化遗产。牵头起草传统村落保护条例，组织制定国家农村生活污水污染物排放标准体系。加强传统建筑工匠培训，推动建设中国传统村落数字博物馆，组织拍摄多部纪录片，深化系列传统村落宣传。

侯轶群，2010届水力学及河流动力学专业硕士毕业，现为水利部中国科学院水工程生态研究所过鱼设施研究室主任。侯轶群长期从事水生态保护与修复研究，重点开展涉水工程生态学效应及减缓措施、涉水工程洄游通道、鱼类基础行为生态规律、江湖连通方案、过鱼设施建设等方面基础和应用研究，为国内20多座大中型过鱼设施项目提供了技术方案、研究成果支撑。

赵梦杰，2015届水利工程专业硕士毕业，现为淮河水利委员会水文局（信息中心）水情气象处副处长、水利部二级水文首席预报员，兼任第八届全国水文专业委员会委员。赵梦杰长期奋战在水旱灾害防御与水利信息化产学研用一线，专注于推动数字技术在水利领域的深度应用，为流域水安全与安徽高质量发展提供坚强技术支撑，助力淮河流域水旱灾害防御工作迈向新台阶。

衷平，2001届环境工程专业本科毕业，现为中国交通建设股份有限公司绿色低碳发展研究中心、碳资产管理中心主任，中交公路规划设计院有限公司副总工程师、绿色低碳部主任。衷平长期从事绿色低碳交通、旅游交通和节能环保方面的规划、研究、设计和咨询等工作。负责和参与国家公路网规划环评、国家交通运输节能环保规划、交旅融合发展重大示范工程研究等20余项重大课题研究及标准制订。衷平领衔的中交公路规划设计院有限公司绿色低碳交通技术攻关团队是中国标准走出去和国际化的重要力量。该团队牵头形成公路工程碳测算、碳定额及碳足迹核算系列标准10余部，完成基建领域首单公路桥梁预制箱梁碳足迹核算，并获得欧洲国际专利；参与建设国家和交通运输部碳计量中心，在交通基建碳计量领域处于领先地位。
