# 孟　新

## 

发布时间：2021-09-15文章来源：河海大学

孟新，女，中共党员，1966年6月生，辽宁开原人，博士，研究员。现任河海大学党委副书记、纪委书记。1990年7月获原南京铁道医学院（现东南大学医学院）临床医学学士学位，在职获得南京大学教育学硕士学位、东南大学法学博士学位。本科毕业后留校工作，历任政治辅导员，分团委书记，校团委书记，东南大学团委书记，浦口校区党工委书记、管委会副主任，生物科学与医学工程学院党委书记，纪律检查委员会副书记、监察处处长（兼），党委组织部部长。2016年2月任河海大学党委副书记、纪委书记。
