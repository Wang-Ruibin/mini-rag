# 河海大学举行校党委中心组集体学习暨树立和践行正确政绩观学习教育读书班第一次专题研讨

## 

发布时间：2026-04-01文章来源：党委宣传部

4月1日，河海大学举行校党委中心组集体学习暨树立和践行正确政绩观学习教育读书班第一次专题研讨。校党委书记杨桂山主持学习，校党委中心组成员参加学习。全体校领导原原本本学习了《习近平关于树立和践行正确政绩观论述摘编》《习近平总书记地方工作期间坚持正确政绩观生动实践》，并结合学习思考和工作实际依次作交流发言，进一步凝聚奋进共识。

会议强调，面对新一轮科技革命、教育强国建设、国家江河战略与国家水网建设带来的重大机遇和现实挑战，要深刻理解把握习近平关于树立和践行正确政绩观的重要论述，对学校坚守治水兴邦使命、以正确政绩观推动高质量发展、服务国家水安全战略，具有十分重要的意义。

会议认为，要坚持党性引领，始终以党性原则校准政绩导向，紧密对接国家重大战略需求，推动学校各项工作始终沿着正确政治方向前进。要坚持高质量发展，以科学决策把准问题为切入口，扎实推动学科发展、教育教学综合改革、人才引育、科研创新、成果转化，不断提升学校办学实力。要坚持站稳师生立场，改进工作理念和作风，聚焦师生急难愁盼问题深化调查研究，在基层治理、服务提质、安全保障等方面持续发力，办好人民满意的教育。要坚持以实干出政绩，秉持“功成不必在我”的境界和“功成必定有我”的担当，全面落实民主集中制和管党治党责任，真抓实干、担当作为，统筹推进学习教育与学校“双一流”建设、“十五五”规划编制实施、年度重点工作落实等，切实将学习成效转化为工作实绩，不断开创世界一流特色研究型大学建设的新局面。
