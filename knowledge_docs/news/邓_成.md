# 邓　成

## 

发布时间：2025-08-27文章来源：河海大学

邓成，男，中共党员，1977年11月生，河南滑县人，研究生学历，工学博士，二级教授，博士生导师，国家级人才计划入选者，享受国务院政府特殊津贴。现任河海大学党委常委、副校长。

2001年7月参加工作，曾任西安电子科技大学电子工程学院副院长、发展规划部部长兼一流建设办公室主任。2025年8月起任河海大学副校长。

主要研究方向为多模态信息处理、人工智能。主持国家重点研发计划、国家自然科学基金重点项目等40余项，获国家自然科学二等奖1项（排序3）、陕西省科学技术一等奖2项（排序1）。
