# 校党委书记杨桂山率团访问新加坡和马来西亚

## 

发布时间：2026-06-22文章来源：国际合作处

为深入推进高水平对外开放办学，深化与东南亚国家高校、科研机构及企业的务实合作，6月12日至17日，校党委书记杨桂山率团赴新加坡、马来西亚开展系列交流访问。

在新加坡国立大学，杨桂山与校长陈永财举行座谈。双方共同回顾两校友好合作历程，就持续深化涉水领域合作、为我校师生搭建更多海外深造与学术交流平台等达成广泛共识。代表团走访了海岸保护与防洪韧性研究所，双方签署共同申报国家留学基金委（CSC）创新型国际化人才培养项目意向协议，为高层次国际化人才联合培养奠定坚实基础，并访问该校数学科学研究所，就学术交流、科研合作等事宜深入交流。

在南洋理工大学，杨桂山与助理副校长杨耀文举行座谈。双方一致同意加快推进本硕双学位、双硕士学位等联合培养项目落地实施，深化师资互访、科研协同与资源共享，推动两校合作提质升级、走深走实。

代表团还受邀出席新加坡国际水周开幕式及水务博览会，学习国际前沿治水理念、关键技术与治理经验，进一步拓展了学校在水务、海岸防护与洪涝治理领域的国际合作网络。

在新访问期间，代表团专程拜访中国驻新加坡大使馆，与教育公参韩骏亲切会见。韩骏表示，大使馆将全力支持河海大学与新加坡高校深化国际交流合作，为各项合作提供有力保障。

在新加坡河海大学校友会代表座谈会上，杨桂山细致询问海外校友工作生活与职业发展情况，勉励广大校友扎根海外、建功立业，持续关心支持母校建设发展，与母校同心同向、携手共进、共促发展。

在马来西亚大学，杨桂山与副校长雅迪玛、助理副校长林爱莲举行座谈。双方重点洽谈本硕双学位联合培养项目，达成在人工智能等领域共建国际联合实验室的意向。代表团走访了该校海洋与地球科学研究所，双方就共同开展国际科研合作、联合申报国际科研项目等深入对接，明确合作路径与推进举措。

在马访问期间，代表团还走访中国港湾有限公司南太平洋区域管理中心等中资企业，围绕深化校企合作、卓越工程师国际化人才培养、企业属地化人才培育等议题充分交流，推动产教融合、科教融汇与国际化人才培养协同发力、落地见效。

此外，代表团在新加坡、马来西亚专项举办海外人才招聘宣讲会，面向新加坡国立大学、南洋理工大学、马来亚大学、马来西亚理工大学等高校优秀青年人才，全面推介学校学科优势、发展平台、师资建设与引才政策，积极吸引海外高层次人才和青年骨干加盟河海。

相关职能部门和学院负责人随团访问。
