# 我校科研团队在沿江化工产业转型与长江大保护研究领域取得新进展

## 

发布时间：2025-05-13文章来源：社科处、公管院

近日，我校公共管理学院朱新华教授团队，依托中国科协战略咨询项目“长江经济带产业绿色转型赋能共同富裕的政策供给研究”，在沿江化工产业转型与长江大保护研究领域取得新研究进展。

该研究探索了沿江化工产业链知识图谱可视化分析新方法。在收集产业关联数据的基础上，构建化工产业链二元矩阵，并利用UCINET工具绘制化工产业链图谱，基于社会网络分析方法的基本原理探究不同行业在产业链中的地位，并结合长江经济带各省市化工产业的相关数据分析长江经济带产业链结构特征，最终归纳出沿江化工产业链存在的问题。

图 化工产业链图谱 

该研究提出了沿江化工产业链韧性转型新思路。以碳中和与碳达峰、长江经济带高质量发展和长江大保护等国家战略为研究切入点，通过对多重战略的政策解读和战略下沿江化工产业所受影响进行研究，借鉴国内外产业转型升级经典案例经验，提出多重国家战略下沿江化工产业转型发展的生产过程与生态环境相协调、产业规模与资源环境容量相适应、产业链与高质量发展要求相匹配和产业布局与自然条件相结合的多维转型路径。

该研究充实了长江大保护战略和产业链供应链韧性的话语体系。沿江化工产业转型是协同长江大保护和提升化工产业链供应链韧性、解决“化工围江”困境的必由之路，研究有助于充实长江大保护战略和产业链供应链韧性话语体系的语料库，为打造具有原创性、引领性、前瞻性中国产业高质量发展和生态保护协同实践的话语体系做出边际贡献。

团队首席专家朱新华教授受聘为中国科协决策咨询专家团队首席专家，团队编制的咨询报告《沿江化工产业转型与高质量发展》2024年7月获第九届高等学校科学研究优秀成果奖（人文社会科学）咨询服务报告类三等奖。该研究的系列成果作为政策建议被上级有关部门和智库采用，并被《学习强国》《新华日报》《南京日报·思想理论》等转载。
