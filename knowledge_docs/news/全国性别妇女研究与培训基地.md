# 全国性别妇女研究与培训基地

## 

发布时间：2025-06-18文章来源：河海大学

河海大学性别与发展研究中心于2010年5月成立，2012年成为中国妇女研究会会员单位。性别与发展研究中心秉承开放、整合和凝聚的精神，旨在通过科研、教学和培训为热爱社会性别研究的广大师生和学者搭建平台。

中心以公共管理学院为主要依托，联合法学院、马克思主义学院、外国语学院等多个学院合作共建，并以科研为中心、以教学为载体，面向社会开展性别培训工作，实现科研、教学和社会实践一体化，理论与实践相结合的工作机制。中心兼职成员主要由各学科热爱社会性别研究的师生、学者组成。
