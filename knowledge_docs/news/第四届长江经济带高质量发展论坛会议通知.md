# 第四届长江经济带高质量发展论坛会议通知

## 

发布时间：2024-11-26文章来源：社科处

为深入学习贯彻党的二十届三中全会精神和习近平总书记关于长江经济带建设的系列重要讲话精神，河海大学、南京农业大学和中国科学院南京地理与湖泊研究所将联合主办第四届长江经济带高质量发展论坛，通过深入交流与探讨，旨在汇聚各方智慧与力量，推动理论创新、实践创新、制度创新和文化创新，为长江经济带的高质量发展贡献新的思路与方案。现将有关事宜通知如下：

一、会议主题

以体制机制创新推动长江经济带高质量发展

二、主办单位

河海大学、南京农业大学、中国科学院南京地理与湖泊研究所

三、会议时间、地点

时间：2024年12月15日8:30—18:00

地点：河海大学西康路校区

四、会议报到

会议报到时间：2024年12月14日14:00—20:00

会议报到地点：隽恒酒店一楼大厅会议报到处

费用：本次论坛不收取会议费，参会人员往返交通费和住宿费自理，住宿酒店建议为南京隽恒酒店或南京瑞迪酒店等。

五、会议回执

请参会人员于2024年12月10日前反馈 “第四届长江经济带高质量发展论坛”会议回执，同时加入“第四届长江经济带高质量发展论坛”微信群。
