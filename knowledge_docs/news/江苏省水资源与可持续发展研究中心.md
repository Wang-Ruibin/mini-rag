# 江苏省水资源与可持续发展研究中心

## 

发布时间：2025-06-18文章来源：河海大学

江苏省水资源与可持续发展研究中心于2009年10月由江苏省教育厅批准设立，是首批江苏高校哲学社会科学重点研究基地。中心以国家战略、服务区域发展、满足行业需求和支撑工商管理学科建设为宗旨，聚焦水资源可持续发展问题，主要依托工商管理学、水利工程、环境科学与工程等学科优势，开展水资源管理及可持续发展的科学研究与人才培养。

主要研究方向包括：（1）水资源优化配置与区域可持续管理；（2）水权水市场与资源可持续管理；（3）水资源高效利用与企业可持续管理；（4）极端天气风险与需水可持续管理。
