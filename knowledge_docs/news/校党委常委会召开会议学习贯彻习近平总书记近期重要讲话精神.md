# 校党委常委会召开会议学习贯彻习近平总书记近期重要讲话精神

## 

发布时间：2026-05-08文章来源：河海大学

5月8日，校党委常委会召开会议，学习习近平总书记近期重要讲话精神、重要回信精神，研究贯彻落实举措。校党委书记杨桂山主持会议并领学。

会议学习了习近平总书记在中央政治局第二十五次集体学习时的重要讲话精神。会议指出，要充分认识做好防灾减灾救灾工作的极端重要性，准确把握我国水旱灾害防御工作面临的新情况新问题。要持续加强应急管理等防灾减灾相关学科建设，依托水灾害防御全国重点实验室，着力扬优势、补短板、强弱项，助力流域防洪工程治理，赋能雨水情监测预报预警体系。要充分发挥安澜-水灾害防御系统的支撑作用，持续提升我国水灾害防御的精准性、时效性和主动性，为全面增强水旱灾害防范应对能力贡献更大力量。

会议学习了习近平总书记在加强基础研究座谈会上的重要讲话精神。会议指出，习近平总书记重要讲话全面分析面临的新形势新挑战，对加强基础研究作出战略部署、提出明确要求，为学校加强基础研究提供了根本遵循。会议强调，要深入学习贯彻习近平总书记重要讲话精神，全力抢抓新一轮科技革命和产业变革历史机遇，切实增强基础研究原始创新的行动自觉。要结合学校“十五五”规划，布局基础学科建设，提升基础学科发展水平，有力支撑学校内涵发展。要持续推进大平台、大团队、大项目、大成果建设，强化基础研究应用牵引，加强有组织科研，推进重大原始创新和关键核心技术协同攻关，更好发挥基础研究主力军和重大科技突破生力军作用，在促进产学研深度融合上实现更多突破。

会议学习了习近平给中国青年五四奖章暨新时代青年先锋奖获奖者代表的回信精神。会议强调，要坚持党建带团建，推进基层团组织标准化、规范化建设，不断建设更加充满活力、更加坚强有力的共青团组织。要强化青年思想政治引领和价值观塑造，深化“大思政课”建设，引导河海青年坚定信念、练就本领，把个人追求融入国家发展大局。要抓实各类条件保障，创造优质育人生态，为青年营造公平公正、干事成才的良好成长环境。
