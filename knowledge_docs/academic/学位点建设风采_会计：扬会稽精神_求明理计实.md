# 学位点建设风采 | 会计：扬会稽精神 求明理计实

## 

发布时间：2026-07-03浏览次数：14

从1915年的簿记学课程起，河海大学会计学位点走过了一条与实业救国、水利兴邦紧密交织的发展之路。一代代河海会计人秉持“艰苦朴素、实事求是、严格要求、勇于探索”的校训精神，将严谨的计量思维融入工程管理，逐步形成了“会计+水利”的鲜明特色，书写着属于自己的 “商业语言”。

萌芽：商科基因植入（1915-1979）

1915年，张謇先生创立河海工程专门学校，率先开设簿记学等商科课程，将严谨的计量思维引入水利工程教育，埋下了河海会计的第一颗种子。1927年，河海工科大学设立商业专修科，商科教育正式纳入学校体系。此后历经时代变迁，1979 年，恢复商学教育，历经动荡岁月的河海商科，在改革开放的春风中重新启航，为会计学专业的后续发展奠定了根基。

成长：专业体系的构建（1985-2010）

从专科到本科，从专业方向到独立系所，河海会计在这一时期开始了系统化发展。1985年起设立财务与会计专科，1991年开设会计学本科（常州），2003年获批会计学硕士点，2010年获工商管理博士点（下设会计学方向）和会计专业硕士学位点（MPAcc），构建起了“本-硕-博”一体化的人才培养体系。

深耕：学位点实力的跃升（2015- ）

从学位授权到国际认证，从省级品牌到国家一流，河海会计的每一步都走得坚实有力。2015年，会计专业硕士学位（MPAcc） 项目通过教育部水平评估，学院同步通过AMBA和PMI-GA国际认证，办学质量获得国际认可。2020年，会计学专业入选国家一流本科专业建设点，2021年，入选江苏省品牌专业并获评校级思政示范专业，专业建设与课程思政双向赋能。2025年，会计硕士专业学位点（MPAcc）参加会计专业学位教育质量认证（AAPEQ），进入建设期，相关工作正在持续推进中，标志着学科在人才培养质量、课程体系建设与行业服务能力方面正朝着全国权威标准稳步迈进。

当前，河海大学会计学位点立足“会计+水利”特色，累计为南水北调、三峡等重大工程输送近千名财会人才，毕业生中超半数服务于财政、税务及水利行业。

学位点深耕水资源资产核算、流域生态补偿、工程资金监管等方向，成果获省部级优秀奖，为流域高质量发展与水利工程资金管理提供了坚实的理论支撑。同时，累计完成智库报告30余项，牵头制定的生态产品价值测算与财务评价模型被纳入地方政策试点，积极服务长三角一体化等国家战略。

面向未来，学位点将主动拥抱数智化变革，深化产教融合，加强与水利行业龙头企业及政府部门的协同育人，着力培养精财务、懂水利、通数据、善实践的高层次复合型会计人才。
