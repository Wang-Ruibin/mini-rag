# 河海大学举行2025年新教师教书育人能力提升培训班（模块一、二）

> 来源: https://jwc.hhu.edu.cn/2025/1114/c3114a310298/page.htm

10月22日，我校举行2025年新教师教书育人能力提升培训班，围绕“课程建设与教学模式改革”和“教育方针政策与教学管理规范”两大模块展开培训。国家教学名师、河海大学陈元芳教授，第五届全国高校教师教学创新大赛一等奖获得者、南京师范大学乐松山教授，第四届全国高校教师教学创新大赛一等奖获得者、苏州科技大学范凌云教授，国家级一流本科课程负责人、河海大学向圆圆副教授，以及教务处副处长朱金秀、陈磊、王玲、陈丹出席活动并作专题报告。我校近一年新入职的140余名专任教师参加培训。活动由朱金秀副处长主持。

 

活动全景

在“课程建设与教学模式改革”模块，陈元芳教授作《工程教育专业认证理念介绍》主题报告，深入阐释了专业认证的内涵与核心理念，引导青年教师将认证理念融入教学实践，推动课程体系与教学设计的持续优化；乐松山教授以《空间数据库课程教学思考与实践》为题，分享了课程建设与教学改革的理论探索与实践过程，为新教师提升课堂教学质量提供指导；范凌云教授在《新文科建设与改革——以<社会综合调查实践>课程创新实践为例》报告中，系统呈现了课程从资源整合到成果落地转化的全流程，为新老师创新教学实践树立了典型范例；向圆圆副教授围绕《“三体系融通，三教法协同”——课程建设经验分享》，详细介绍了一流课程的建设路径与实践经验，为青年老师开展课程建设带来有益参考。

在“教育方针政策与教学管理规范”模块，朱金秀、陈磊、王玲、陈丹四位副处长，分别就教师教学发展、教研评估建设、教务管理运行、实践教学体系等工作进行政策解读，帮助新教师全面了解学校教学管理制度，引导其规范教学行为，提升教学质量，鼓励大家积极投身学校建设及个人成长，共同助力学校高质量发展。

为提升我校本科教学质量与水平，科学、规范、有效地开展我校新教师教书育人能力提升工作，自2020年度开始，教务处通过系统的模块化培训课程，帮助教师熟悉新时代高等教育理念与方针政策，掌握先进教学方法和手段，提升教学技能和教学创新素质，促进教师教学综合能力养成，为切实提高教学质量奠定坚实基础。
