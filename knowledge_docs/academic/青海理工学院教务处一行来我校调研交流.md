# 青海理工学院教务处一行来我校调研交流

> 来源: https://jwc.hhu.edu.cn/2026/0417/c3114a324835/page.htm

4月17日，青海理工学院教务处处长陈慧媛一行来我校调研交流。我校教务处处长李轶，副处长陈磊、王玲及部分科室代表参加了座谈会。会议由李轶处长主持。

会上，李轶处长对青海理工学院教务处领导和同仁的到来表示热烈欢迎，并就学校办学概况与人才培养工作作了简要介绍。

在讨论环节，双方围绕专业设置、交叉学科微专业建设、课程教学改革、质量保障、讲课竞赛及本科生转专业等方面进行了深入的交流探讨，并就前期青海理工学院选派赴我校交流学生的学习生活情况进行了细致沟通。通过本次深入交流，双方在教学研究、人才培养等方面凝聚广泛共识，并就依托“一带一路”教育合作平台深化务实合作达成一致，为两校持续深化交流合作、实现高质量协同发展奠定了坚实基础。
