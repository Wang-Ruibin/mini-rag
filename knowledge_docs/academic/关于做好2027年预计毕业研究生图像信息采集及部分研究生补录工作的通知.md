# 关于做好2027年预计毕业研究生图像信息采集及部分研究生补录工作的通知

## 

发布时间：2026-04-20浏览次数：2536

各学院（系）：

为严格落实教育部相关要求，根据《关于做好我省2026年高校毕业生图像信息采集工作的通知》（苏教指通〔2026〕7号）要求，我校委托江苏省高校招生就业指导服务中心（以下简称“省招就”）统一采集2027年预计毕业研究生电子图像并补录部分研究生图像信息。现将有关事宜通知如下：

一、采集时间

**1.** **正常采集阶段**

**202****6****年4月****21****日****9:00****-****4月27****日****17:00**

2. **不合格照片重新采集阶段**

**2026年4月28日9:00-4月29日17:00**

**注意：**学校会审核并重置不合格照片，请学生关注采集平台消息。

二、采集对象

1.所有2027年预计毕业研究生

2022级且学制5年、2023级且学制4年、2024级且学制3年、2025级且学制2年的研究生。

2.今年预计毕业且未采集图像的研究生

请登录“研究生教育教学管理系统”，确认自己是否无学历学位照片。

三、采集办法

步骤1：学生提前登录中国高等教育学生信息网（简称“学信网”），获取专属加密二维码（简称“采集码”），后续小程序将对采集码进行识别解密，用于图像传输。操作流程参照附件1。

**注意：**部分学生因休学等原因无法下载采集码；待学籍恢复正常后，参与下一次图像采集。

步骤2：学生登录微信，搜索小程序“江苏图采”或关注“江苏省招就中心”公众号，点击“服务指南”中“图像采集”项目，实名验证后，按步骤操作，操作流程参照附件2。

四、采集要求

1.光线：**建议面向光源，自然光拍摄（不要背对窗户，不要让灯光打在头顶上方）**；确保**面部光线均匀**。

2.衣着：**不得穿蓝色上衣**（后期照片为蓝色背景）**，建议是****白色****上衣**，避免复杂图案。

3.姿态与表情：姿态端正，表情自然，双眼平视，嘴唇自然闭合。

4.眼镜：常戴眼镜者应佩戴眼镜，但不得戴有色(隐形)眼镜，**眼镜不能反光。**当无法避免眼镜反光时请摘下眼镜。

5.佩饰及遮挡物：不得佩戴耳环、项链等，头发不得遮挡眉毛、耳朵，不宜化妆。

6.严禁翻拍：注重真实性，严禁翻拍照片，严禁利用软件美颜处理，**如有违规，学生账号将被锁定**。

五、收费标准

经报省物价主管部门备案，省招就将为学生提供电子图像采集、实名验证、后期制作、排版冲印、统一寄送等服务；为每位学生提供电子版照片和6张纸质照片；收费标准为15元/人（纸质照片和电子图像由省招就统一交付学校）。

六、注意事项

1.学生拍摄检测通过后，照片将自动存入草稿箱，草稿箱可储存6张，学生可选择一张最满意的照片上传；若均不满意，经冷静期后再拍摄，直到满意为止。

**该照片将用于毕业证书、学位证书等，****请务必拍到自己认为最满意的照片后再点击支付****。图像采集结束后，若因个人拍照原因造成图像有瑕疵，不组织重拍。**

2.学校会按照要求对照片进行定期审核，将不合格照片重置并将原因反馈给学生，请学生关注采集平台。

3.若需要图像采集的预计毕业生登录平台后无采集数据，请将学号、姓名、性别、学院、专业信息发给所在学院（系）研究生秘书。
