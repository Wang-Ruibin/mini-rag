# 河海大学2025年新教师教书育人能力提升培训班开班

> 来源: https://jwc.hhu.edu.cn/2025/0928/c3114a307932/page.htm

9月26日，河海大学2025年新教师教书育人能力提升培训班顺利开班。副校长沈扬出席培训班并讲话，教务处处长李轶作专题报告，学生处处长王方，宣传部副部长姜海霞，发规处副处长李伟玲，研究生院副院长王森林，科技处副处长孙林，社科处副处长李轮，人资处副处长汤鸣鸿参加培训班，电气院王楚扬副教授作经验分享。我校近一年新入职的专任教师140余人参加了本次培训。培训由教务处副处长朱金秀主持。

 

活动全景

沈扬副校长向各位新加入河海大学的教师致以热烈欢迎与诚挚祝贺。他强调，教师作为“强国建设、民族复兴”的核心力量，肩负着知识传播、灵魂塑造与新人培育的时代使命。他结合自身教学发展经历，对新教师提出三点期望。一是秉持初心，立德树人。聚焦服务国家战略与新质生产力发展需求，牢记教书育人初心使命，聚焦学校人才培养总目标，努力成为学生成长的引路人。二是勤奋学习，赋能成长。树立终身学习理念，提升数字赋能教学与跨学科融合能力，注重教学创新与专业成长。三是团结协作，共谋发展。主动融入学科团队和教学集体，多向资深教师请教治学经验与育人方法，以团队协作助推本科人才培养质量迭代升级。

 

沈扬副校长讲话

教务处处长李轶作《新时代本科教学改革》专题报告。他结合当前高等教育发展趋势，从国家战略需求、学校发展定位和教师育人使命三个维度，系统阐述了我校本科教学改革的顶层设计与实施路径。同时，融合教学理论与实际，为新教师理解教学政策、把握教学方向、提升育人能力提供了清晰指引，助力新教师将个人成长融入教育强国建设，切实担当起为党育人、为国育才的主力军使命。

 

李轶处长作专题报告

培训班特邀电气院王楚扬副教授作经验分享，他结合个人成长经历，围绕新教师职业初期发展难题，分享了其踏实教学、扎实科研，并切实将教学和科研成果服务于本科人才培养的策略与体会，鼓励新教师立足自身领域持续积累，在长期深耕中实现综合能力跃升，为其快速融入育人角色提供了具有参考价值的实践路径。

 

王楚扬副教授作经验分享

为提升我校本科教学质量与水平，科学、规范、有效地开展我校新教师教书育人能力提升工作，自2020年度开始，教务处通过系统的模块化培训课程，帮助教师熟悉新时代高等教育理念与方针政策，掌握先进教学方法和手段，提升教学技能和教学创新素质，促进教师教学综合能力养成，为切实提高教学质量奠定坚实基础。
