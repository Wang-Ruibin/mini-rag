# 我校播音与主持艺术、英语专业接受长三角新文科教育专业认证现场考查

> 来源: https://jwc.hhu.edu.cn/2025/1107/c3114a309800/page.htm

11月2日至4日，长三角新文科教育专业认证联盟专家组对河海大学播音与主持艺术、英语专业开展联合进校考查。专家组组长由上海交通大学陈业新教授、华东理工大学张先梅教授共同担任，来自江南大学、南京师范大学、上海财经大学等高校及企业的专家参加考查。河海大学副校长沈扬，相关职能部门代表，公共管理学院、外国语学院党政班子及专业师生代表出席现场考查活动。

11月3日上午，专业认证现场考查见面会在河海大学江宁校区举行。联合考查组组长陈业新教授介绍了此次现场考查的专家组成员、考查目的与内容。播音与主持艺术、英语专业负责人分别汇报了专业本科人才培养情况及其特色与优势。在质询环节，专家组成员结合前期线上听课和自评报告等材料审查情况，围绕教师教学能力提升、教学投入激励机制、课程体系建设、招生与就业等方面，与学校相关职能部门和学院负责人进行了深入交流。

现场考查期间，专家组查阅课程试卷、毕业论文等教学资料，调阅教学管理机制文件等支撑材料，实地考察学院实验室等硬件设施，与教务处、学院负责人、专任教师一对一座谈，与在校生、毕业生及用人单位代表访谈等，通过多种形式开展了全面而深入的现场考查。

11月4日下午，专家组召开认证联合进校考查意见反馈会。播音与主持艺术专业专家组组长陈业新教授和英语专业专家组组长张先梅教授分别对标《新文科教育专业认证标准》，对2个专业整体考查情况和考查意见进行了逐一反馈。专家组成员结合个人现场考查情况和切身体会，交流了个人考查意见。专家组充分肯定我校播音与主持艺术、英语专业紧扣时代发展趋势与区域发展需求，不断优化培养目标定位，强化课程思政建设，注重学生综合素质提升，健全质量保障长效机制，人才培养质量得到学生、用人单位和社会的广泛认可与高度评价。同时表示，专业在优化培养方案设置、提升教师数字素养、深化校企交流合作、加强毕业生跟踪机制建设、提高国际视野等方面仍有一定的改进空间。公共管理学院党委书记吴宝海和外国语学院院长张海榕分别作表态发言，表示将认真吸收专家组意见，进一步完善和改进专业建设。

河海大学副校长沈扬代表学校对长三角新文科教育专业认证联盟及各位专家的辛勤付出和诚恳指导表示感谢。他指出，专家组的反馈意见为学校推进文科教育教学改革、优化人才培养体系提供了重要遵循，学校将以此次认证为契机，以点带面持续推进新文科专业建设的内涵与水平，优化专业结构布局，深化学科交叉融合，探索微专业建设路径，完善质量保障体系建设，着力培养更多适应未来社会多元化需求、具有家国情怀和国际视野的高素质复合型文科人才。

长三角新文科教育专业认证联盟是推动我国新文科自主专业认证试点的重要平台，联盟自2020年成立以来，始终坚持高标准、高要求，构建第三方专业认证和学校自主专业认证相结合的双轮驱动式质量保障体系，引导专业遵循“学生中心、产出导向、持续改进”的认证理念，追求卓越质量文化，积极开展新文科建设。我校法学专业已于2023年顺利通过该联盟认证，成为江苏省首个通过长三角新文科教育专业认证的试点专业，为省内高校新文科专业建设提供了有益借鉴。目前，联盟已有来自上海、江苏、浙江、安徽等50余所高校加盟，形成了跨区域协同推进新文科质量提升的良好格局，为培养新时代高素质文科人才提供有力支撑。

河海大学播音与主持艺术专业设立于2008年，入选江苏省一流本科专业，江苏省特色专业。作为省内唯一开设该专业的部属院校，专业推行以实践教学为主的小班化教学模式，近年来，获江苏省普通高等学校本科优秀毕业论文2篇，国家级创训项目8项，指导学科竞赛省级及以上获奖380余项。学生继续深造率约46%，国家级媒体签约率达30%，多名毕业生就职于中央广播电视总台、新华通讯社、人民日报社、湖南广电、江苏广电等各级广播电视台，以及爱奇艺、芒果TV、字节跳动等新媒体。

河海大学英语专业2021年入选国家一流本科专业建设点，2022年、2024年连续入选江苏高校品牌专业建设工程二期、三期项目。专业推动“外语+”复合型人才培养改革，2025年设立“全球胜任力与国际水生态文明”创新班，开设“全球胜任力与数字生态文明建设”微专业。近年来，入选国家级一流本科课程1门，教育部首批新文科项目1项、产学合作协同育人项目4项，指导学科竞赛获省级及以上奖项50余项。学生继续深造率达45%以上，毕业生就业率稳定在95%以上，入选外交部定向培养计划，就职于江苏省外事翻译中心、中江国际集团、苏美达集团等企事业单位。
