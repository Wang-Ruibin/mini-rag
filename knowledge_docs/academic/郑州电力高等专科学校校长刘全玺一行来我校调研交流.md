# 郑州电力高等专科学校校长刘全玺一行来我校调研交流

> 来源: https://jwc.hhu.edu.cn/2025/0808/c3114a306199/page.htm

8月7日，河海大学校友、郑州电力高等专科学校校长刘全玺一行来我校调研交流。我校教务处副处长陈磊及部分科室代表参加了座谈会。会议由陈磊副处长主持。

会上，陈磊副处长对郑州电力高等专科学校校长刘全玺、教务处处长李文举、许多老师的到来表示热烈欢迎，并简要介绍了学校本科教学工作的整体情况和特色亮点。刘全玺对母校的热情接待表示感谢，同时介绍了郑州电力高等专科学校的概况，并表示希望借鉴母校在本科教学方面好的经验和做法，进一步推进自身建设。在讨论环节，双方就人才培养方案、教师课堂教学质量评价、人工智能通识课程建设、教学管理平台建设、本科教学制度建设等方面进行了深入交流。
