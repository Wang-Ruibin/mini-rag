# 港航院“五位一体”工作体系驱动内涵式发展

> 来源: https://jwc.hhu.edu.cn/2022/0810/c3147a239608/page.htm

在学校第十三次党代会以来，港航学院党委以习近平新时代中国特色社会主义思想为指导，在党建引领、立德树人、队伍建设、科学研究、文化建设等五个方面下功夫，高质量赋能学院内涵式发展。

**党建引领促发展，构建“大党建”工作格局**

高举“一面旗”，全面开展“五好”党委领导班子建设。执行党委会会议议事规则、党政联席会议议事规则，保证工作机制好。党史学习教育计划执行到位，党支部书记例会坚持到位，保证思想作风好。制定学院党委委员联系支部办法，制定教代会、专业委员会、学术分委员会等议事办法，保证民主决策好。开展党委委员、班子成员主题培训，“共产党人精神谱系”主题研讨等，开展班子务虚会、教授研讨会等，保证科学发展好。党委书记带头参加评定检查考核，带头对班子成员进行廉政谈话，带头参加系所民主生活会，班子成员每年通过全院大会报告个人有关事项、述职述廉，保证清正廉洁好。

拧成“一股绳”，全面开展高质量基层建设。坚持校、院、支部、党员联动，领导班子成员联系党支部指导工作，切实把党员组织起来、把人才聚集起来、把群众动员起来。学院党委定期召开支部书记例会，配备书籍和设备建设“党员之家”和“支部书记工作室”，建设系所文化走廊。通过优化完善各基层党支部与江苏高等学校优秀科技创新团队建设协同共进的路径，形成学术团队党支部建设工作新模式。学院先后获评全国党建工作标杆院系、江苏省教育系统先进集体、首批全省党建工作样板支部培育建设单位等荣誉，涌现出团中央活力团支部、全国高校“双带头人”教师党支部书记工作室、江苏省工人先锋号、江苏省高校先进基层党组织等优秀集体和个人。获评江苏省党建创新二等奖、最佳党日活动优胜奖共计4项。

下活“一盘棋”，全面开展科研思政建设。贯彻落实全国教育大会精神，创新提出“科研思政”概念，将政治与业务关联，凸显专业特色，依托党员教育实境课堂（严恺陈列室）开展专项研讨，组织江苏省优秀党员等杰出教师宣讲科技报国事迹，将“一带一路”倡议、“海洋强国”和“生态文明建设”等国家战略融入科研学术方向，注重培养师生的历史责任感和爱国主义情怀。在科研工作中严格要求，举办学术道德讲座，坚持学术成果政治审查，做科研立德为先，做学问树人为要，出成果前言为重，达到政治与业务的双丰收。

**立德树人秉初心，培育“大视野”时代新人**

思想为“本”，强化信念引领。学院团委围绕“建党100周年”等主题，组建“张闻天精神调研团”，承办四史知识竞赛活动，提升理论学习能力。开展团干部工作坊，指导策划团日活动，激发基层支部活力。打造“学海导航”、“港航青年说”、“同舟课堂”三大讲堂，打造以教师指导、校友交流、朋辈辅导为主体的学院品牌活动，形成“三全育人”合力。学院团委获评江苏省五四红旗团委。获评全国高校活力团支部1个和江苏省先进班集体5个，2个团支部被推荐为江苏省“百强千优”团支部培育。10人次获评“全国水利院校优秀毕业生”等国省级荣誉。1人获“江苏省团干部大比武”二等奖。

教学为“要”，深化教学改革。探索一套深化课程思政改革方案。加强党支部对基层教学组织建设的指导作用，以点带面，遴选专业核心课程开展校级课程思政示范课程建设，通过学院专题座谈、校友走访调研、行业专家指导，将思政元素融入课程大纲，将校友经历、行业前沿与治水报国情怀融入课程设计。打造一批一流课程教学教材资源库。3门课程入选国省级一流本科课程，1门课程入选省级一流本科虚拟仿真实验课程，2本教材入选江苏省高等学校重点教材。形成一批高水平教育教学成果。学院获中国高教学会优秀成果奖2项、江苏省教学成果奖3项，指导学生获省优秀博士学位论文4篇、江苏省本科优秀毕业设计（论文）5项、江苏省优秀团队2项。

能力为“重”，开展多维育人。组织“学海争锋”学术论坛、国际研讨学术交流，拓宽学术视野，强化学生科研能力。加强校企合作，选派学生到对口单位实践实训，与上海航道局开设“岗前培训班”，探索育人新模式。与5家单位建立志愿服务基地，打造学校十佳志愿服务项目，组织学生定期维护“望归圃”，让学生尊重劳动，弘扬劳动精神。以赛促创，进一步推动人才培养机制创新，学生获省部级以上级学科竞赛奖励200余项，获“挑战杯”全国大学生课外学术科技作品竞赛二等奖，江苏省红色专项赛道一等奖、二等奖各1项，“互联网+”大学生创新创业大赛江苏省赛铜奖1项。

**教师发展严作风，构建“大先生”成长体系**

加强师德师风建设，塑造教书育人新风范。将师德考核贯穿教师管理和职业发展全过程，制定教职工年终考核方案、教师党支部书记“双带头人”培育工程实施方案等，完善支部书记的评价考核细则，进一步规范党员教师的聘期考核体制。积极推进“双创”支部的建设工作，将党支部建设为教师理论学习、考核监督、典型宣传的坚实阵地。1个党支部入选“全国党建工作样板支部”培育创建单位，1个党支部申报全省党建工作样板支部培育创建对象，2个党支部、1个教师工作室入选河海大学“强基创优”培育创建项目。

树立“暖心工程”品牌，搭建传承式进阶体系。学院针对青年教师开展基金申请、备课讲课、生涯规划等帮扶，举办多期青年教师论坛，强化供给侧发力。强化中生代教师的示范作用，学院定期邀请杰出教师分享学生培养和科研学术的心得，以党支部为单位开展经验分享与定点指导的“传帮带”活动。在老教师退休之际，学院组织“政治生日”组织生活会，举办简朴而隆重的欢送仪式，邀请退休教师凝练传递几十年工作经验。近五年，获宝钢优秀讲师奖3人，全国研究生教育工作先进个人2人，江苏省教育工作先进个人1人。2人获全国水利类专业青年教师讲课竞赛特等奖、2人获评全国大学生水利创新设计大赛优秀指导教师、20人次获评“高教杯”全国大学生先进成图技术与产品信息建模创新大赛优秀指导教师。21人次入选长江学者、国家杰青、万人计划等国家级人才计划，13人次入选省级人才计划。获得全国首批“111”引智计划项目，为青年俊才培养提供了平台渠道。以2个国家级平台、3个省部级平台、1个创新中心为支撑，入选省部级创新团队3支。

**科学研究立潮头，践行“大工程”伟大使命**

推进一流学科建设，打造前沿性科研平台。学院重点打造江苏省沿海海洋水温观测站网、大型波浪港池实验量测系统、L型风浪流港池系统、河口海岸风暴潮与沿海城市多源水灾害预报预警系统等四个教学科研平台，其中L型风浪流港池系统建成后将达到国内顶尖、国际领先水平。增设“船舶与海洋工程”一级学科硕士学位授权点、“海洋技术”二级学科博士学位授权点以及“土木水利”、“资源与环境”博士专业学位授权点，进一步优化学科布局。

培育一流科研团队，深度服务国家重大战略。学院把握专业优势，深化社会服务，组织师生参与港珠澳大桥、长江口深水航道等大国工程建设，牵头江苏省908专项，建立教育部海洋野外观测站，获批江苏省军民融合创新平台。近五年来，获得57个国家自然科学基金项目资助，经费为3234万；新增科研项目578项，新增科研合同经费2.5亿元；获得83项中央高校业务费项目资助。被SCI收录论文451篇，被EI收录论文148篇，出版21部专著。授权发明专利296项，授权实用新型专利169项，批准登记软件著作权50项。获省部级以上奖励33项。

**文化建设永向前，推动“大文化”传承赋新**

赓续百年红色基因。学院党委牢牢把握“学党史、悟思想、办实事、开新局”的工作目标，将党史学习教育与学院历史传承、水利精神弘扬紧密结合，高质量高标准组织开展党史学习教育百余项，深入挖掘党史学习教育的文化底蕴。学院连续多年获评河海大学精神文明先进单位。

涵养特色廉洁文化。学院连续七年举办廉洁伞书画活动、连续六年举办诚信扇书画活动，将传统文化和反腐倡廉、诚信教育相结合，有效地推动了校园党风廉政建设与诚信主题教育工作，营造了风清气正的校园氛围，获评校园文化建设优秀成果奖，并在江苏省第十一届文化艺术节上作为精品活动推广。

传承优秀港航文化。以申报江苏省党员教育实境课堂示范点为契机，精心打造严恺陈列室党员教育实境课堂，传承港航争先精神。邀请全国劳模夏风等一批杰出校友回到母校，在其奋斗经历的宣讲中弘扬港航文化。成立“海样青年”学生社团，承办全国涉海高校慢跑公益活动；承办全国海洋知识竞赛选拔赛，组织参加全国总决赛。
