# 河海大学2026年江苏高校品牌专业建设工程二期项目验收材料公示

> 来源: https://jwc.hhu.edu.cn/2026/0602/c3115a327555/page.htm

根据《关于做好江苏高校品牌专业建设工程二期项目验收工作的通知》要求，现对《江苏高校品牌专业建设工程二期项目立项建设专业验收汇总表》和《江苏高校品牌专业建设工程二期项目立项建设专业验收报告》进行公示。 

    公示时间：2026年6月2日至6月8日。如有异议，请以书面形式反馈至教务处教研科。联系电话：025-58099177。
