# 河海大学召开2026年本科教学重点工作布置会

> 来源: https://jwc.hhu.edu.cn/2026/0312/c3114a323511/page.htm

3月9日，河海大学召开2026年本科教学重点工作布置会，副校长沈扬出席会议。学生处分管领导，各学院本科教学院长、教学秘书以及教务处和双创学院全体人员参加会议。

教务处李轶处长对2026年本科教学重点工作做出总体安排，教务处相关领导分别就教学发展、教研评估、教务学籍、实践教学等方面工作内涵进行了细化解读。双创学院孙梦雅副院长介绍了2026年创新创业重点工作，学生处王璐副处长通报了近期学生就业工作进展。水文院任黎副院长、土木院吴建涛副院长、计算机院张鹏程副院长、农工院张洁副院长、机电院苑明海副院长、智能院胡银龙副院长、新能源院程亚军副院长、外语院王璐副院长分别围绕本学院特色工作进行了交流。

沈扬副校长作总结讲话并就本学期重点工作提出要求。他强调，一是强化协同育人机制，扎实做好四个联动。深入推进专业学科联动、本科生与研究生培养联动、教学与学工联动、招生培养就业全链条联动，打破壁垒、整合资源，构建全方位协同育人格局。二是加快专业转型升级，持续深化产教融合。通过优化专业结构、打造产教融合特色班、推进微专业建设，全面提升专业内涵与核心竞争力，增强人才培养适配度。三是紧盯关键核心指标，抓实重点建设任务。稳步推进智慧课程建设，推动人工智能与教育教学深度融合，强化优质教材建设、高水平教学竞赛及学生“三大赛”组织，切实以赛促教、以赛促创、以赛促学。四是夯实教学基础底线，从严抓好日常教学管理。严格落实教学规范，强化过程监控与质量保障，保障教学秩序平稳有序、教学运行高效规范，持续夯实人才培养质量根基。
