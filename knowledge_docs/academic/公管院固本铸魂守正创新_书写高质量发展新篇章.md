# 公管院固本铸魂守正创新 书写高质量发展新篇章

> 来源: https://jwc.hhu.edu.cn/2022/0902/c3147a240122/page.htm

自学校第十三次党代会召开以来，公共管理学院党委坚持以习近平新时代中国特色社会主义思想为指导，全面贯彻党的十九大和十九届历次全会精神、全国教育大会、全国思想政治工作会议等精神，坚持立德树人根本任务，胸怀“国之大者”，以服务支撑国家重大决策和重点战略需求为导向，切实担负起为党育人为国育才使命，以高质量党建引领学院高质量发展，为助力学校“双一流”建设、中华民族伟大复兴而不懈奋斗。

**党建引领，深化双融双促，“党建+学科”有高度。**充分发挥党建引领作用，以师生“小”支部为基点，构建“大”党建格局。以“党建+信仰——学思践悟有提升”“党建+学科——教学科研有促进”“党建+文化——价值传递有情怀”为党支部建设核心主线，提出“理论学习会+学术研讨会”双会联开新模式，将学术调研活动与党日活动相结合，实现业务工作与党建工作结合、学术研究与党性提升相结合。移民研究中心党支部、学院党委党日活动获江苏省2020、2021年度最佳党日活动优胜奖，“咏诵党课”获评江苏省信仰公开课省级示范党课，被新华社、扬子晚报等主流媒体报道。战“疫”专题微党课、特色案例，被推荐至教育部“四个一”行动，实现了支部建设与学科发展“双提质”，教工党建与学生党建“双促进”，教师队伍与青年学子“双成长”。

**开拓创新，找准发展路径，学科建设有深度。**将“特色”作为凝练学科方向的重要标准，将树立“四个服务”的意识作为凝练学科方向的总体要求，主动服务国家和人民需求，积极探索选择更具潜力和前景的发展路径，充分聚集优秀人才、挖掘优势资源、创设优质环境。2017年获批社会学一级学科博士点、心理学一级学科硕士点；2019年获批新闻与传播专业学位硕士点；2021年获批移民科学与工程交叉学科。社会学、公共管理学科获评江苏省重点学科，积极完成公共管理一级学科博士点的申报工作。在全国第四轮学科评估中，社会学学科获评B，进入前30%；公共管理学科获评B-，进入前40%。

**匠心育人，不忘使命担当，“三全”育人有厚度。**大力优化课程体系、完善课程教学，全方位挖掘课程思政元素，将社会主义核心价值观融入课程、教材和教学全过程。劳动与社会保障专业、土地资源管理专业获评国家一流本科专业、江苏省品牌专业；社会学专业、播音与主持艺术专业获评江苏省一流本科专业。劳动与社会保障专业稳居A类专业行列，土地资源管理专业首次进入A类，社会学、广播电视学专业位居B+类。入选江苏省研究生教育教学改革课题1项，校级课程思政重点项目4项，一般项目13项。获江苏省优秀博士论文1篇，江苏省优秀硕士论文8篇，江苏省优秀本科毕业论文3篇，省优团队1个。入选江苏省研究生科研创新计划项目27项，江苏省研究生实践创新计划项目34项。

**固本铸魂，牢记“国之大者”，思政工作有热度。**以立德树人为根本，以培育和践行社会主义核心价值观为主线，构建课堂教学、校园文化、社会实践、网络空间四位一体的育人格局，把立德树人融入思想道德、文化知识、社会实践教育各环节，以“四三一”模式全面提升学院思想政治工作质量。深入建设学院品牌项目“青年思想者论坛”“梦想公开课”“学海争锋”等文化育人项目，获第十七届“挑战杯”全国大学生课外学术科技作品竞赛全国一等奖1项、江苏省二等奖1项、红色专项赛道一等奖1项、第七届中国国际“互联网+”大学生创新创业大赛产业命题赛道江苏省三等奖2项等。职业生涯规划等项目助力学校取得突破，连续三年获得江苏省职业生涯规划大赛特等奖、优秀指导教师称号。全力打造“有高度、有营养、有情怀、有文化”的“四有”网络平台，传递公管人文精神。学院官方微信公众号“河海思语”累计推送1665篇，阅读量86.3万，营造出文明、健康、积极向上的文化氛围。

**多措并举，响应国家需求，科学研究有温度。**锚定国家战略需求，强化资源经济与政策、环境社会学、移民科学与管理等研究领域，拓展应急管理、老年社会保障等新兴前沿交叉领域。科研项目合同金额累计1.168亿元，获国家基金资助项目41项，其中国家社科基金重大项目1项、重点项目5项，国家自然科学基金项目重点项目1项。获省部级科研成果奖24项，其中省社科奖13项、省部级科学技术奖11项。发表SSCI检索论文108篇、SCI检索论文71篇、CSSCI论文374篇，出版专著37部。获批教育部国别与区域研究中心——亚洲研究中心、江苏高校哲学社会科学重点建设基地——环境与社会研究中心。《关于江苏省新型冠状病毒性肺炎疫情防控和社会治理的建议》等，获省委书记吴政隆、副省长陈星莺的批示和采纳。《推进沿江化工产业高质量发展的对策建议》等多篇决策咨询报告获全国人大常委会委员、农工党中央副主席杨震重要批示。

**乘势而上，坚持“引”“育”结合，队伍建设有力度。**优化学院人才引育流程，制定科学合理使用人才的措施，全面加强师资队伍建设，以创新团队、学术带头人和中青年优秀教师为重点，全力打造师德高尚、规模适当、结构优化的教师队伍，引进专任教师20名，其中具有博士学位19名，具有海外留学经历者10名，具有高级职称教师8名。全力支持青年教师快速成长，新增江苏省“青蓝工程”中青年学术带头人1名、江苏省第五期“333工程”第三层次培养对象1名、江苏省紫金社科优青人选4名，校“大禹学者计划”15名、“青年教授”3名。通过开展“师德模范”评选、新老教师结对帮扶、师德师风警示教育等活动，引导教师以成为“大先生”为目标，弘扬尊师重教、崇尚科学、勇于创新的良好风尚。

**紧跟时代，拓宽合作领域，国际交流有广度。**留学生培养体系逐步完善，建成6个全英文授课专业，1门留学生校级精品课程，1门全英文线上品牌课程。共招收留学研究生186名，派出高水平公派项目研究生5名、自费留学本科生129名。新增3位外籍兼职教授，派出教师13名赴国外知名高校访学进修，邀请外籍专家80余人次来院做讲座，累计50余人次出国参加重要国际会议并汇报论文，累计23人次在国际水电协会、国际移民网络等国际组织任重要职务，另有2名教师任国际期刊编委。学院与印度行政管理学院签署研究生交流、科研合作、学术交流和教师互访合作框架协议。主办第二届国际移民与社会发展学术研讨会暨国际水电移民对话高端论坛和国际移民网络2019年会。
