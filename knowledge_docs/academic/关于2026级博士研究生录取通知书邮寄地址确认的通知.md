# 关于2026级博士研究生录取通知书邮寄地址确认的通知

## 

发布时间：2026-07-01浏览次数：11

各位拟录取考生：

为保证录取通知书及时、准确送达，请有关考生（硕博连读第一批除外）登录河海大学研究生招生管理系统（https://yzss.hhu.edu.cn/logon）确认博士研究生录取通知书邮寄地址、邮政编码及联系方式，招生项目选择“**博士研究生****招生考试**”，登录账号为**报名号**、密码为注册时填写的密码（**直博生和联培生**登录账号为**考生编号**、初始密码为身份证号后六位），选择模块“通讯地址确认”。系统开放截止时间为**202****6****年****7****月****6****日****下午****14:00**，逾期将按系统默认地址进行寄送，本系统仅开放一次，请及时完成确认工作。

拟录取考生经教育部批准后方可正式录取。博士研究生录取通知书预计将于7月中下旬根据考生思想政治素质和品德考核、调档（仅非定向研究生）等情况陆续发放，请考生及时关注研究生院网站通知。
