# 河海大学教师在第五届全国高校教师教学创新大赛中获佳绩

> 来源: https://jwc.hhu.edu.cn/2025/0819/c3114a306356/page.htm

近日，由教育部高等教育司指导，中国高等教育学会主办的第五届全国高校教师教学创新大赛决赛在北京举办。我校代表江苏参赛的三位教师（团队）表现优异，**共斩获两项二等奖和一项三等奖**。其中，**机电院陈润楚团队（课程《设计心理学》）**获新文科中级及以下组二等奖，**土木院李宗京团队（课程《钢结构》）**获课程思政中级及以下组二等奖，**港航院刘琳团队（课程《水利工程制图》）**获新工科中级及以下组三等奖。

 

河海大学3位获奖教师

（左起依次为刘琳、陈润楚、李宗京）

 

获奖课程聚焦学科特色与教学改革前沿，充分彰显了我校长期深耕教学改革，在推动“四新”建设、深化课程思政改革、促进学科交叉融合方面的综合实力和显著成效。**《设计心理学》**课程突破传统学科界限，构建了“艺工融合、数字赋能”的特色教学体系，打造“虚实结合、项目驱动”教学模式，使学生能够将心理学理论与设计实践融会贯通，切实提升跨学科创新思维和解决复杂问题的能力。**《钢结构》**课程围绕国家重大基础设施建设，深度挖掘专业教学中的思政育人元素，深耕课程思政创新实践，有效实现专业知识传授与价值塑造的有机融合，为培养服务国家战略需求的高素质专业人才提供了可复制的教学模式。**《水利工程制图》**课程立足新工科建设需求，以“数字赋能水利人才培养”为核心，将前沿技术与创新方法融入传统基础课教学，实现理论知识与教学实践深度融合，为培养水利领域拔尖创新人才提供有力支撑。

长期以来，学校始终坚持本科教学核心地位，高度重视教师教学能力提升，通过搭建“国-省-校”三级联动的教师教学竞赛培养机制，推动教学创新成果向常态化课堂转化，形成“以赛促教、以赛促改、以赛促学”的良性生态。备赛期间，教务处全程提供保障，组建校内外高水平专家辅导团，通过多轮次精准辅导、模拟演练和成果凝练，为参赛教师提供强有力的支撑。

全国高校教师教学创新大赛是两个最高水平教学竞赛之一。本届大赛共设新工科、新农科、新医科、新文科、基础课程、课程思政、产教融合七个赛道。赛事规模再创新高，全国32个赛区1308所高校的34160名教师参加省赛，588个教师（团队）进入全国决赛。最终共评选出全国一等奖88项、二等奖205项、三等奖295项。
