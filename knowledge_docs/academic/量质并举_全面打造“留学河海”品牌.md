# 量质并举 全面打造“留学河海”品牌

> 来源: https://jwc.hhu.edu.cn/2022/0810/c3147a239609/page.htm

留学事业历来与国家和民族的命运紧密相连，世界各国因此形成了激烈的人才竞争格局。2018年，习近平在全国教育大会上指出：将我国建成全球主要留学中心和世界杰出青年向往的留学目的地。要扩大教育开放，同世界一流资源开展高水平合作办学。近年来，国家有战略，河海有行动，学校自十三次党代会以来，以国际化推进工作会议精神为引领，面向国际、精准聚焦，通过构建四大体系，实现了留学生教育的新突破，有力地提升了学校的国际知名度。

**构建优质生源体系 不断优化留学生规模结构**

学校始终以“扩大规模、优化结构、规范管理、保证质量”为工作方针，坚持并突出学校办学特色。在改进高校国际交流合作评价等方面常抓不懈，推进留学生教育一体化发展、趋同化培养、规范化管理。在疫情常态化背景的严峻形势下，学校通过构建网上申请、网上审核、网上面试、网上录取的全链条管理流程，改版升级中英文招生网站，创新招生宣传模式，2021年录取学历生408人，为历史最高。学校积极争取国家部委的支持，精准聚焦服务国家外交战略和“一带一路”倡议。学校会同教育部组织实施“‘澜湄合作’水资源高层次人才计划”，“西非国家港口、海岸及近海工程高层次人才培养计划”，“面向非洲的杰出女性电气工程师项目”；其中“‘澜湄合作’水资源高层次人才计划”获评教育部“2019-2020学年‘丝绸之路’中国政府奖学金优秀项目”和“2021-2022学年‘丝绸之路’项目优先支持项目”（全国18个），获批中国政府奖学金生自主招生计划数位列江苏高校前茅。会同教育部、水利部实施“一带一路”水利高层次人才项目；学校会同江苏省组织实施“‘一带一路’西非国家土木工程高层次人才项目”、“‘一带一路’环境科学与工程高层次人才项目”和“百名优才留学河海计划”，为葡语国家培养专门人才；学校加强与中国电建集团华东院、澜湄水资源合作中心、中国水科院、水利部农村电气化研究所等行业领军单位的联系沟通，拓展生源渠道，开发新项目，确保可持续发展。经过多年努力，留学生生源质量不断提升。学校已实现三个校区留学生教育协同开展，成功入选留学江苏目标学校。2020-2021年，学校经受了“疫情大考”，稳住了留学生教育基本盘。2021年学校留学生规模1423人，从2014年的全省第15位上升至第5位，其中硕士留学生规模全省第1，博士留学生规模全省第2，中国政府奖学金学生全省第2，显示了良好的结构层次。

**构建质量保障体系 不断强化留学生培养能力**

学校重视专业和课程建设，发挥教学团队的作用，精心制定培养方案，加强过程管理，在本、硕、博都设有全英文授课专业和课程，多元质量保障体系吸引越来越多的外国学生前来留学，成为“留学河海”重要魅力源。学校重视培养模式的创新，积极助力行业海外发展，与中国电建集团、中国交通集团、中国长江三峡集团等行业龙头企业深化合作，共建留学生实习实践基地，定制化培养留学生，服务行业企业海外工程人力资源本土化。与上述企业协同培养留学生取得可喜的成绩，得到了上级主管部门和行业单位的广泛支持与认可。近3年来，学校新增3门国家级来华留学精品课程，新增20门省级来华留学精品课程，新建365门校级来华留学精品课程，新建来华留学生英文教材24部，新建留学生实习实践基地15家，专业和课程建设不断取得新的突破。学校高质量通过教育部组织的中国政府奖学金绩效评价，“面向澜湄合作机制框架的国际学生培养模式创新与实践”获评2021年江苏省教学成果奖一等奖。

学校贯彻落实习近平总书记和党中央有关统筹疫情防控和教育高质量发展的重要部署，确保安全稳定，2020年疫情以来南京和常州两地三校区的疫情防控工作布置周密。学校认真落实《来华留学生高等教育质量规范（试行）》，持续推进留学生培养质量工程，完善留学生培养制度，修订留学生培养方案，精心组织线上线下教学，围绕品牌专业、英文教材、精品课程等丰富“留学河海”品牌内涵，2020至2021年，学校共开设留学生线上课程503门次，修课人数共计6320人次，校领导深入网络课堂，了解督导线上教学工作。

学校发挥孔子学院中方院校办学主体作用，做好孔子学院的转隶工作，努力拓展孔子学院建设。新建孔院中方外派教师库。法国阿尔萨斯孔子学院克服疫情影响，继续保持平稳发展，内部管理稳定，中外方配合得当，教学与文化活动齐头并进，在法国东部地区尤其是阿尔萨斯地区享有较高的知名度和良好口碑。设6个教学点，开设中国语言、文化课程等20余门课程，注册学员达228人，各类班级22个。举办并协办弘扬中华文化的各类活动。

**构建制度优化体系 不断提高留学生管理水平**

近年来，学校不断加强留学生教育调研研究，在招生、培养、管理、联合办学等方面出台了3项校级管理制度和多项管理流程，同时梳理修订了自13年以来相关工作制度，进一步明确了工作职责，理顺工作关系，提高管理水平。学校申报获批教育部备案的澜湄国家区域研究中心，在人才培养、科学研究、资政服务、国际合作交流与国际教育等方面搭建了高层次平台。学校在三校区联合出入境管理部门定期分别召开留学生法制教育培训会，联合公安、外事、教育等部门进行安全巡查，建立与保卫、后勤、物业等单位的留学生宿舍联席会议制度，经常走访学生宿舍，掌握和了解师生思想状况情况，及时处置潜在风险。从未发生重大安全事故和网络负面舆情。“河海大学高质量留学生培养，助力‘一带一路’”被学校推荐为教育部直属高校教育2018年重点工作奋进之笔。新华社、中新社、中央电视台、光明日报、江苏卫视、新华日报等主流媒体集中报道河海大学留学生教育工作。学校2017年当选江苏省留学生教育管理研究会理事长单位，并于2021年连任。学校连续六年被江苏省教育厅、公安厅、省外事办公室、江苏留管会评为江苏省来华留学生教育先进集体。

**构建素质拓展体系 不断提升留学生综合素养**

学校服务国家战略，承办教育部、水利部和江苏省“学在中国来华留学博士生论坛”“庆祝新中国成立70周年长三角高校国际学生江苏行”“江苏省人才地图暨首届留学生专场招聘会”“一带一路留学生感知中国水利”等重要活动。学校大力开展“感知中国”“感知江苏”“感知河海”“感知水利”等感知系列品牌活动，受到国家留学基金委、省教育厅、行业单位和南京市外办的高度评价。申办“感知中国 运河史诗” 获评国家留学基金委品牌项目和品牌基地，学校积极引导留学生发现“世界美、中国美、江苏美、河海美”为主题开展丰富的感知品牌文化活动。开展“感知中国·丝路之源”活动，组织留学生代表赴陕西石泉考察，与石泉县金蚕小镇和中国电建集团贵阳院共建河海大学留学生实习实践基地。组织留学生广泛开展学术交流、文化艺术、“感知中国”等活动，提供跨文化交流的平台，促进中外学生友好交流。学校组织留学生在侵华日军南京大屠杀遇难同胞纪念馆开展“珍爱和平 开创未来”和平教育主题活动。留学生在长三角区域高校学生跨文化能力大赛、长三角高校外国留学生才艺展演、“一带一路”沿线国家青年论坛、“一带一路”国际青年户外挑战赛等活动中展示了优秀的综合素质和良好形象，在“友谊是一首歌——在江苏外国人唱中文歌曲大赛”中获得二等奖。学校在2019年和2021年先后举办首届和第二届澜湄水资源合作青年论坛活动，得到广泛参与和一致好评。

我们将始终以党的领导为坚强保障，以服务国家战略为导向，以十四五事业发展规划目标为动力，以培养造就更多优秀人才为任务，以质量与特色为核心竞争力，紧跟学校双一流建设步伐，更加精准聚焦、更加奋发有为，以教育评价改革为导向，持续推进留学生教育提质增效，以优异的成绩喜迎学校第十四次党代会的胜利召开！（国教院）
