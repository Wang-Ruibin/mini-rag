# 信息学部全面助力学校“双一流”建设

> 来源: https://jwc.hhu.edu.cn/2022/0812/c3147a239621/page.htm

我校信息学部成立以来，传承创新，全面贯彻落实学校十三次党代会的要求，探索新机制、新模式与新方法，始终保持良好发展势头，深入学习贯彻习近平新时代中国特色社会主义思想和党的十九大精神，全面贯彻落实党的教育方针，牢牢把握社会主义办学方向，加强党的全面领导、全面从严治党，落实立德树人根本任务，完成了“十三五”党建和思想政治工作规划各项任务，学部的综合实力迈上新台阶。

**发挥党建政治引领作用 营造良好发展环境**

学部党委深入贯彻落实中央要求，发挥党建政治引领作用，全面深入开展了“两学一做”学习教育，“不忘初心、牢记使命”主题教育，党史学习教育，师生的党性意识明显增强，党支部的战斗堡垒作用得到较好发挥，获评“江苏省党建工作样板支部”、“江苏省优质党支部”各1个，河海大学最佳党日活动1项，河海大学“强基创优”党建工作样板支部项目1项。

信息学部党委成立以来，深化党政共同负责的领导体制和工作机制，完善党政联席会议以及党委会等会议制度，发挥“三重一大”决策机制以及民主集中制的组织原则，加强领导班子作风建设，强化“四个意识”，不断提高领导班子的凝聚力、向心力。更加突出强调党的群众工作的重要性，将党建工作向基层延伸，上下监督与沟通机制畅通，营造了良好的发展环境。严格标准，突出强调规范化流程，积极发展党员，目前教工党员比例占教工总数的65%； 学生党员占比21%。

**健全思政工作体系 培养拔尖创新人才**

信息学部认真落实全国高校思政工作会议精神，将思想政治教育要求渗透到人才培养的各个环节。先后新增智能科学与技术、软件工程、人工智能三个本科专业；持续推进教育教学改革，创新人才培养模式，“十三五”以来，获得国家级教改课题9项，省部级教改课题8项，省教学成果奖3项，计算机科学与技术专业入选国家“一流本科建设”计划，“云计算技术与应用”入选国家级线上线下混合式一流课程。

提升学生创新能力，出台政策激励师生参加各级学科竞赛，Ⅰ类竞赛获奖累计100余项，两次获得“创青春”大学生创业大赛全国金奖；涌现出以江苏省大学生年度人物冯全同学为代表的大学生创业典型。制定大学生创新创业教育孵化平台建设与管理规定，整合资源，建立校内导师指导的管理机制，校外实践实训基地，强化日常训练，增强大学生动手实践能力，搭建各渠道多项目的储备。获第七届中国国际“互联网+”大赛全国银奖1项，江苏省一等奖1项，产业命题赛道大赛二等奖1项，第十七届“挑战杯”全国大学生课外学术科技作品竞赛红色专项江苏省二等奖1项。加强就业指导工作，3个项目入选教育部第一期供需对接就业育人项目。培养优秀典型，陈德龙同学以“艺”抗“疫”，策划组织宁汉两地高校学子“云合奏”，获2019年江苏省大学生年度人物提名奖。继续深造升学率37%，学位率95%以上，就业率95%以上。研究生培养质量稳步提高。获批“江苏省研究生培养创新工程”项目35项，省研究生教育教学改革研究与实践课题4项，获省优秀硕士学位论文8篇，研究生优质生源率34%，就业率98%以上。

学工团队成立“青春计E”网络思政工作室，贴近大学生思想、学习、工作和生活的实际需求，开展富于创造性，具有推广价值和示范意义的实践工作。成立“喂！友·with you”心灵驿站，立足于广大学生的基本需求开展工作，做到“慰忧”“为优”“微友”三位一体，为学生疏解忧愁，为学生的未来成长保驾护航。 

**优化学科布局 提升学科整体实力**

信息学部成立以来，整合优化计算机与信息学院与物联网工程学院学科，形成学科发展合力，学科群布局初步形成。学部紧跟国家与社会需求，立足长远发展，做好科学规划，同时加强平台建设，为学科发展提供了较好的科研平台。目前，计算机科学成功进入ESI全球学科排名前1%；信息与通信工程学科获批博士后流动站；软件工程学科高质量通过省高校优势学科建设工程二期验收并成功入选三期建设项目；完成计算机科学与技术、软件工程、信息与通信工程和电子科学与技术四个一级学科教育部两轮（第四轮，第五轮）评估；抢抓发展机遇，牵头组织“人工智能”交叉学科博士点增设工作，人工智能学院揭牌正式成立。2021年，学部新增水利部水利大数据重点实验室一个部省级科研平台，水利学科与信息学科结合，服务国家需求，为学校“双一流”建设提供了有力支撑。

**推动机制体制改革 提升科研与社会服务水平**

结合新时代科研评价体系改革导向和“破五唯”要求，学部修订完善科技成果评价体系，强化科学研究质量与贡献导向，深化科研评价改革工作，建立学科特色和研究性质相适应的科学评价体系，以科研团队组建以及重要科研平台建设为依托，加强成果凝练，促进重大科技成果产出。强化引导和支持，提高论文成果质量，推动产出重大原创性基础研究成果。 瞄准国家需求、行业需求、区域需求、企业需求，主动对接，加强协同，积聚科学研究与社会服务持续发展的能力。十三五以来，科技成果显著，累计科研经费合同额近2亿，到账额1.5亿，累计成果转化67项。获得国家自然科学基金项目61项，其中国家重点研发计划项目和重点项目各1项，面上项目31项，国家级科技奖励1项，省级科技奖励44项，高水平论文（sci）800余篇，授权专利近800项。

**优化师资队伍 培养领军人才**

以学科、专业、课程团队为牵引，分层次规划和建设师资队伍。通过年度绩效考核、聘期考核评定，建立较为规范的激励机制；通过整合学科方向、创建新专业等，为教师搭建事业发展平台；通过设立专项资助，大力支持教师教学科研成果产出、国际合作交流。逐步朝构建重教学、强科研、与办学定位相适应为目标的师资队伍方向努力。坚持引育并举，“十三五”期间引进成熟人才14人，引进青年博士23人；专任教师博士比达到80%。其中获得省部级有突出贡献中青年专家、江苏省“333高层次人才工程”、教育部“新世纪优秀人才支持计划”、宝钢教学名师奖、江苏省“六大人才高峰”、江苏省高等学校“青蓝工程”培养对象等各类省部级人才计划及荣誉近30名。学工团队获评立项教育部人文社科研究项目1项、2021年度江苏省高校哲学社会科学思政专项1项、江苏省教育厅思政专项项目1项。获评江苏省辅导员工作案例一等奖1项，江苏省高校网络教育优秀作品工作案例一等奖1项，“第一届’三人行’”新学工论坛”优秀工作案例一等奖2篇。

**打开发展格局 拓展国际交流**

为学习先进理念与管理方式，同时提升科研水平，学部积极开展展与世界高水平大学、科研机构的深层次合作，着力提升合作层次，促进合作形式多样化。“十三五”以来，与新加坡南洋理工大学、德国比勒菲尔德应用技术大学、美国密苏里科技大学、台湾宜兰大学等国（境）外高校开展交换生、短期访学等项目。与国外高校实验室签订科研合作协议；举办海峡两岸“智慧城市建设”论坛、“人工智能+”等高水平国际学术会议；获批1项国家重点研发计划政府间国际科技创新合作重点专项（韩国），国际合作科研项目2项；国际联合发表论文100余篇；获批江苏省外专百人计划1项。在校留学生91人，全英文专业课程建设17项；选派出国（境外）交流本科生120余人次。

改革创新永远在路上，艰苦奋斗永远不停歇，学部党委和行政将继续全面履行政治责任、强化思想引领、落实保证监督、推动改革发展，带领广大师生攻坚克难、落实中央精神和上级决策部署，团结凝聚师生，推动学部取得更大的发展和更优异的成绩！（信息学部）
