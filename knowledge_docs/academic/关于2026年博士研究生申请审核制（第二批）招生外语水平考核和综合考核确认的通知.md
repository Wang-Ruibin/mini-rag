# 关于2026年博士研究生申请审核制（第二批）招生外语水平考核和综合考核确认的通知

## 

发布时间：2026-05-26浏览次数：13

各有关考生：

根据我校博士研究生招生工作安排，现将我校2026年博士研究生申请审核制（第二批）招生外语水平考核、综合考核考前确认相关安排通知如下。

一、参加外语水平考核对象

各单位官网公布的2026年博士研究生申请审核制（第二批）材料审核通过须参加外语水平考核的考生（各单位陆续公布中）。

二、外语水平考核时间及地点

（一）考试形式：笔试

（二）考试时间：2026年6月6日（周六）上午9:00-11:00

（三）考试地点：河海大学西康路校区（南京市鼓楼区西康路1号）

三、综合考核时间及地点

（一）考试形式：面试

（二）考试时间：2026年6月6日-7日，具体以各单位通知为准

（三）考试地点：以各单位通知为准

四、同等学力加试时间及地点

（一）考试形式：笔试

（二）考试时间：

2026年6月7日上午9:00-11:00，政治理论

2026年6月7日下午13:00-15:00，加试业务课一

2026年6月7日下午16:00-18:00，加试业务课二

（三）考试地点：河海大学西康路校区（南京市鼓楼区西康路1号）

五、考前确认

**外语水平免试考生可直接进行综合考核确认；需参加外语水平考核考生则先进行外语水平考核确认，待考核通过后再进行综合考核确认（时间另行通知）。**

**（一）确认时间**

请于北京时间2026年6月3日上午10:00前（具体以各单位通知为准）完成相关考核确认，逾期无法操作。打印外语水平考核、综合考核准考证不受以上时间限制，参加考核前下载打印即可。

**（二）确认内容**

**具体确认内容及其他注意事项请查阅《关于2026年博士研究生申请审核制招生外语水平考核和综合考核确认的通知》****（网址：https://gs.hhu.edu.cn/2026/0330/c17277a324076/page.htm）。**
