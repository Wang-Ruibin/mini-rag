# 土木院推动党建业务融合促进高质量发展

> 来源: https://jwc.hhu.edu.cn/2022/1020/c3147a247455/page.htm

近年来，土木与交通学院党委认真落实学校党委的决策部署，全面落实立德树人根本任务，大力推动学院党的建设与人才培养、队伍建设、科学研究、国际合作等深度融合，促进学院各项事业高质量发展。

**坚持固本强基，提升学院党委引领发展能力**

持续强化理论武装。学院党委坚持以习近平新时代中国特色社会主义思想为指导，坚决贯彻执行党中央和上级党组织的重要决策部署，扎实推进党史学习教育，把学习成果转化为推动发展的实际成效。

坚持党政共同负责。坚持民主集中制，制定完善学院《学院党委会议制度实施细则》《学院党政联席会议制度实施细则》，通过党委会、党政联席会议，集体研究决定学院工作中的重要事项，将院党委的政治责任融入到办院治学全过程各环节，把握好教学科研管理等重大事项的政治原则、政治立场、政治方向，以改革创新、勇于担当和团结务实的精神,助力事业发展，为建设一流研究型学院凝心聚力、保驾护航。

激发教师党支部活力。以“双带头人”为引领，强化教师党支部的政治功能和组织功能，促进教师党支部党建与业务的深度融合。全面贯彻落实立德树人根本任务，把教师党支部建设与人才全过程培养贯通起来，推动党建与课程思政深度融合，实现组织育人和课程育人“双线推进、一体融合”。学院2个教师党支部入选江苏省首批党建工作样板支部培育建设单位，1个支部被认定为江苏高校特色党支部，1个支部被认定为江苏省高校先进基层党组织，获批校级党支部书记工作室1个。

发挥榜样力量，进一步发挥教师党员在育人过程中的带头作用。深入挖掘周氐等老教授和良渚申遗水利团队的先进事迹，营造崇尚师德师风的良好氛围。院长高玉峰教授领衔的“土木工程防灾减灾教师团队”成功入选教育部第二批“全国高校黄大年式教师团队”。徐卫亚教授带领的“岩石力学与防灾减灾”导师团队荣获第三届江苏省“十佳研究生导师团队”称号，高玉峰教授荣获“十佳研究生导师”称号。

**坚持为党育人，加强大学生思想政治工作**

学院坚持聚焦立德树人根本任务，以大力实施“四项工程”为抓手，实现全员全过程全方位育人。

强化思想引领，筑牢党团磐石工程：坚持党建带团建，形成纵横覆盖的堡垒，搭建梯度式理论学习平台，打造“土木交通青年说”品牌活动，与中国建筑、上海建工、苏交科等大型国有企业开展党团共建活动。学院团委获得江苏省“五四红旗团委”的荣誉称号。

涵育优良学风，夯实学风建设护航工程：以学业规划为中心，营造一个氛围、构建学业困难帮扶和学生能力提升两个体系，组建课程辅导团、朋辈讲师团、专家导师团三支队伍，开展“土木大讲堂”“土木微讲堂”等活动，选聘优秀的党务工作者、青年教师、优秀学生作为专业导师、学业导师、兼职辅导员，1名辅导员获得江苏省辅导员年度人物。

推进学科竞赛，开拓创新创业远航工程：搭建全链条式创新创业教育培育体系，建立土木类创客空间、大学生科创中心、创新创业社团，实现覆盖全员、全过程的个性化薪火传承式培养，获得“挑战杯”大学生课外学术科技作品竞赛国赛二等奖1项，省赛赛特等奖1项、红色专项特等奖1项、一等奖1项；“挑战杯”中国大学生创业计划竞赛国赛银奖1项，省赛银奖1项、铜奖1项；“互联网+”江苏省大学生创新创业大赛二等奖1项、三等奖5项。

探究精准就业，充实生涯规划续航工程：探索专业教育融入生涯规划教育，搭建“生涯启蒙、素质培养、生涯决策、职业发展”多维平台，学院获批成立职业生涯规划工作室，获得省职规赛创新创业赛道总冠军1项和研究生赛道特等奖1项。构建“课程、讲座、个性方案、情境实习”体系，实施推进校企合作、打造品牌活动等举措，引导学生服务国家战略、服务西部建设，实现毕业生精准、高质、对位就业。

**坚持党管人才，提高师资队伍建设水平**

学院党委牢固树立“人才资源是第一资源”的理念，加强师资队伍建设顶层设计，建立与人才培养和学科发展相适应的师资队伍引育规划和年度实施方案，并以党政联席会决议形式加以确定，成为学院党政的共同意志和共同责任。五年学院引进新教师近40人，其中副教授以上人员超过三分之二。入选国家级人才项目6人，省部级人才项目20人次。

重视青年教师发展，夯实学院发展后劲。学院140名教师中，“80后”占比超过80%。院党委大力弘扬“宽厚务实，通达致远”的文化精神，关心、关注和支持青年教师成长。建立院领导联系青年教师制度，院长主动担任院党委青年委员，密切与青年教师的交流沟通，指导他们的学术发展。建立青年教师发展档案，为年轻教师特别是海外归国教师配备“师傅”。成立青年教师联谊会，指导分工会定期举办青年教师学术交流会。鼓励优秀青年教师担任党支部书记和支部委员，促进青年人才“双提高“和”双发展“。近五年，学院入选国家青年人才计划项目5人。

**坚持党建融合，推动学院全面发展**

学院坚持党建引领，凝聚奋进力量，把党建优势转化成创新优势、发展优势。土木工程学科稳步前进。在教育部第四轮学科评估中获评A-，排名并列全国第七，通过江苏高校优势学科建设工程三期项目建设，获评优秀，2021年位列软科世界一流学科排名第51-75名；本科教学成果丰硕。获2021年江苏省教学成果奖特等奖1项，国家首批新工科研究与实践项目验收优秀，获批教育部首批虚拟教研室建设试点。入选国家级一流课程、国家级课程思政示范课程及国家级精品资源共享课等10多门，入选国家首届优秀教材奖1部、国家精品教材奖1部。入选江苏省青蓝工程优秀教学团队和江苏高校优秀基层教学组织各1支。专业课程思政体系成为江苏课程思政示范专业指标建构蓝本。学院本科教学质量保障体系载体《河海大学土木类本科教学系列蓝皮书》在全国31个省份253所高校推广；研究生培养质量稳定提高。教育部博士学位论文与江苏省硕士学位论文抽检合格率连续多年保持100%；科技创新水平不断提高。五年累计新增科研合同经费41868万元，新增国家自然基金立项113项，获国家或省部级科技奖励58项，其中作为牵头单位获国家科技进步奖二等奖1项，发表高水平论文成果1388篇，授权国家发明专利764项，实现科研成果转移转化收益累计680余万元；国际合作与交流持续深化。与加拿大多伦多大学、法国里尔科技大学、佐治亚理工学院、西澳大利亚大学、香港科技大学等国际知名高校的合作进一步加强。近五年，累计招收培养境外留学生96名，长期国家公派研究生84人；资助师生参加国际学术交流107人次，新增国家“111计划”引智基地1个、江苏省国际联合实验室1个。
