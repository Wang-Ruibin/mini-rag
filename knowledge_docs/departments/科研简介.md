# 科研简介

> 来源: https://math.hhu.edu.cn/kxyj/list.htm

在科研方面，长期以来坚持“入主流、强特色”的发展理念，在不断加强数学基础研究的同时，十分注重服务学校优势学科的交叉研究，先后形成了5个相对稳定的研究领域：微分方程与动力系统、代数与组合数学、高性能计算、概率统计与数据科学、数学与其他学科交叉。近5年主持1项国家重点研发计划子课题，获批国家自然科学基金项目41项，省部级项目21项，发表高水平SCI论文300余篇。

在交叉研究方面，我们有着优良的传统，老一辈就开展了结合实际工程问题的有限元和最优化的数学理论和应用研究，在曲线拟合坐标下有限元的构建、弯曲边界问题的四阶计算、混合有限元的有效算法、环保问题中的水质规划、黄河中下游水资源优化及水量调度等方面取得了有效的研究成果，服务了工程实际。

新生代传承了老一辈的传统，围绕国家水安全，融入到学校生态水利、智慧水利、数字孪生等水利科技创新项目中，提供数学的重要支撑，取得了一些成果。参与获得教育部科技进步一等奖1项、黄河水利委员会黄河水利科学研究院科学技术进步奖一等奖1项、水力发电科学技术奖二等奖1项等多项涉水奖项。近2年主持的单项经费50万以上的横向科研项目就有5个。
